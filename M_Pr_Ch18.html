Bivariate Normal Distribution

The bivariate normal is the natural generalisation of the bell curve to two dimensions. It is the gateway to multivariate statistics, machine learning, and portfolio theory. This chapter derives its PDF from matrix principles, proves its marginals are normal, finds the conditional distributions, and establishes the linear combination result.

Five Parameters

$(X,Y) \sim \text{BVN}(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$ with:

$\mu_1 = E(X)$, $\mu_2 = E(Y)$ — the means
$\sigma_1^2 = \text{Var}(X)$, $\sigma_2^2 = \text{Var}(Y)$ — the marginal variances
$\rho = \text{Cor}(X,Y) \in (-1,+1)$ — the correlation coefficient

$\rho = 0$: $X$ and $Y$ are independent. $|\rho| = 1$: $X$ and $Y$ are linearly related.

The PDF — Derived from Univariate Normal

Start with the univariate Normal PDF and rewrite it in matrix form. This reveals exactly how to generalise to two dimensions.

Univariate in matrix form: $f(x) = \frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$ can be written as $(2\pi)^{-1/2}|\Sigma|^{-1/2}\exp\!\left(-\frac{1}{2}(x-\mu)\Sigma^{-1}(x-\mu)\right)$ where $\Sigma = \sigma^2$.

Bivariate generalisation: replace $\sigma^2$ with the $2\times 2$ variance-covariance matrix: $$\Sigma = \begin{pmatrix}\sigma_1^2 & \rho\sigma_1\sigma_2 \\ \rho\sigma_1\sigma_2 & \sigma_2^2\end{pmatrix}$$ replace $(x-\mu)$ with the vector $(x-\mu_1, y-\mu_2)$, and $\sigma^2 = |\Sigma|^{1/2}$ with $|\Sigma|^{1/2}$.
Computing $|\Sigma|$ and $\Sigma^{-1}$: $$|\Sigma| = \sigma_1^2\sigma_2^2 - \rho^2\sigma_1^2\sigma_2^2 = \sigma_1^2\sigma_2^2(1-\rho^2) \implies |\Sigma|^{1/2} = \sigma_1\sigma_2\sqrt{1-\rho^2}$$ $$\Sigma^{-1} = \frac{1}{|\Sigma|}\begin{pmatrix}\sigma_2^2 & -\rho\sigma_1\sigma_2 \\ -\rho\sigma_1\sigma_2 & \sigma_1^2\end{pmatrix} = \frac{1}{1-\rho^2}\begin{pmatrix}1/\sigma_1^2 & -\rho/(\sigma_1\sigma_2) \\ -\rho/(\sigma_1\sigma_2) & 1/\sigma_2^2\end{pmatrix}$$ The quadratic form in the exponent: $$(x-\mu_1,\, y-\mu_2)\,\Sigma^{-1}\begin{pmatrix}x-\mu_1\\y-\mu_2\end{pmatrix} = \frac{1}{1-\rho^2}\left[\frac{(x-\mu_1)^2}{\sigma_1^2} - \frac{2\rho(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2}\right]$$
BVN PDF: $$f(x,y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}}\exp\!\left(-\frac{1}{2(1-\rho^2)}\left[\frac{(x-\mu_1)^2}{\sigma_1^2} - \frac{2\rho(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2}\right]\right)$$ When $\rho = 0$: this reduces to $f_X(x)\cdot f_Y(y)$ — the product of two independent Normals. ✓
Shape: $f(x,y)$ is a 3D bell surface. Level curves $\{f(x,y) = C\}$ are ellipses, oriented by the covariance. When $\rho=0$ and $\sigma_1=\sigma_2$, the ellipses become circles. Positive $\rho$ tilts the ellipse northeast; negative $\rho$ tilts it northwest.

MVN generalisation: For $n$ variables, $f(\mathbf{x}) = (2\pi)^{-n/2}|\Sigma|^{-1/2}\exp\!\left(-\frac{1}{2}(\mathbf{x}-\boldsymbol{\mu})^T\Sigma^{-1}(\mathbf{x}-\boldsymbol{\mu})\right)$.

Marginal Distributions

If $(X,Y) \sim \text{BVN}$, then $X \sim N(\mu_1, \sigma_1^2)$ and $Y \sim N(\mu_2, \sigma_2^2)$.
To find $f_X(x) = \int_{-\infty}^\infty f(x,y)\,dy$, complete the square in $y$ inside the exponent: $$\frac{(x-\mu_1)^2}{\sigma_1^2} - \frac{2\rho(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2}$$ $$= (1-\rho^2)\frac{(x-\mu_1)^2}{\sigma_1^2} + \left(\frac{y-\mu_2}{\sigma_2} - \rho\frac{x-\mu_1}{\sigma_1}\right)^2$$ Substitute $z = (y-\mu_2)/\sigma_2$; the second square becomes $\left(z - \rho\frac{x-\mu_1}{\sigma_1}\right)^2$. Then substitute $u = \frac{z - \rho(x-\mu_1)/\sigma_1}{\sqrt{1-\rho^2}}$: $$f_X(x) = \frac{1}{\sqrt{2\pi}\sigma_1}e^{-\frac{(x-\mu_1)^2}{2\sigma_1^2}} \cdot \underbrace{\frac{1}{\sqrt{2\pi}}\int_{-\infty}^\infty e^{-u^2/2}\,du}_{=1} = \frac{1}{\sqrt{2\pi}\sigma_1}e^{-\frac{(x-\mu_1)^2}{2\sigma_1^2}} \qquad \square$$
The converse is false. If $X \sim N(\mu_1,\sigma_1^2)$ and $Y \sim N(\mu_2,\sigma_2^2)$ separately, the joint distribution need not be BVN. Marginal normality does not imply joint normality. This is a common error in applied statistics.

Conditional Distributions

$$X|Y=y \;\sim\; N\!\left(\mu_1 + \rho\frac{\sigma_1}{\sigma_2}(y-\mu_2),\;\; \sigma_1^2(1-\rho^2)\right)$$ $$Y|X=x \;\sim\; N\!\left(\mu_2 + \rho\frac{\sigma_2}{\sigma_1}(x-\mu_1),\;\; \sigma_2^2(1-\rho^2)\right)$$
Two things to notice:

1. The conditional mean is linear. $E(X|Y=y) = \mu_1 + \rho\frac{\sigma_1}{\sigma_2}(y-\mu_2)$. This is the regression line of $X$ on $Y$. The slope is $\rho\sigma_1/\sigma_2$.

2. The conditional variance is reduced. $\text{Var}(X|Y=y) = \sigma_1^2(1-\rho^2)$. Knowing $Y$ shrinks the uncertainty in $X$ by a factor of $(1-\rho^2)$. Perfect correlation ($|\rho|=1$): zero conditional variance — $X$ is determined exactly by $Y$. Zero correlation ($\rho=0$): conditional variance = marginal variance — $Y$ tells us nothing about $X$.

MGF and Linear Combinations

MGF: $$M_{X,Y}(t_1,t_2) = \exp\!\left(\mu_1 t_1 + \mu_2 t_2 + \tfrac{1}{2}(\sigma_1^2 t_1^2 + 2\rho\sigma_1\sigma_2 t_1 t_2 + \sigma_2^2 t_2^2)\right)$$ Compare with $M_X(t) = e^{\mu t + \frac{1}{2}\sigma^2 t^2}$ — the same structure with $t_1, t_2$ playing the roles of $t$, and the covariance term $2\rho\sigma_1\sigma_2 t_1 t_2$ appearing in the cross term.

Linear combination: If $(X,Y)\sim\text{BVN}$ and $a,b\neq 0$: $$aX + bY \;\sim\; N\!\left(a\mu_1 + b\mu_2,\;\; a^2\sigma_1^2 + b^2\sigma_2^2 + 2ab\rho\sigma_1\sigma_2\right)$$ Proof: set $t_1 = at$ and $t_2 = bt$ in the joint MGF.
In portfolio theory: if Nifty return $X \sim N(\mu_1,\sigma_1^2)$ and Bank Nifty return $Y \sim N(\mu_2,\sigma_2^2)$ with correlation $\rho$, then a portfolio $Z = aX + bY$ has $Z \sim N(a\mu_1+b\mu_2,\; a^2\sigma_1^2 + b^2\sigma_2^2 + 2ab\rho\sigma_1\sigma_2)$. This is the Markowitz mean-variance framework in one line. The cross term $2ab\rho\sigma_1\sigma_2$ is why diversification reduces variance when $\rho < 1$.

probability / ch18 / q01★ Easy

$(X,Y) \sim \text{BVN}(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$. What is the marginal distribution of $X$?

A
$N(\mu_1, \sigma_1^2\sigma_2^2)$
B
$N(\mu_1, \sigma_1^2)$ — same as if $Y$ didn't exist
C
$N(\mu_1, \sigma_1^2(1-\rho^2))$
D
Depends on $\rho$
Answer: B — $N(\mu_1, \sigma_1^2)$

The marginal of $X$ is obtained by integrating $f(x,y)$ over all $y$. The result is $N(\mu_1,\sigma_1^2)$ regardless of $\rho$. The correlation affects the joint structure but not the individual marginals. Option C is the conditional variance $\text{Var}(X|Y=y)$, not the marginal.
probability / ch18 / q02★★ Medium

$(X,Y)\sim\text{BVN}$ with $\rho=0.6$, $\sigma_1^2=4$. What is $\text{Var}(X|Y=y)$?

A
$4$ — conditioning doesn't change variance in BVN
B
$4 \times 0.6^2 = 1.44$
C
$4(1-0.6^2) = 4 \times 0.64 = 2.56$
D
$4/0.6^2 \approx 11.1$
Answer: C — $2.56$

$\text{Var}(X|Y=y) = \sigma_1^2(1-\rho^2) = 4(1-0.36) = 4 \times 0.64 = 2.56$. Knowing $Y$ reduces the variance of $X$ by the factor $(1-\rho^2)$. With $\rho=0.6$: 36% of the variance is "explained" by $Y$, and 64% remains. This is the coefficient of determination $R^2 = \rho^2$ in disguise.
probability / ch18 / q03★★ Medium

$X \sim N(0,1)$ and $Y \sim N(0,1)$ separately. Can we conclude $(X,Y) \sim \text{BVN}$?

A
Yes — two Normal marginals always give BVN
B
Yes — if they're standard Normal, the joint must be standard BVN
C
No — marginal normality does not imply joint BVN
D
Only if $\rho = 0$
Answer: C

The converse of the marginal theorem is false. There exist joint distributions with Normal marginals that are not BVN. The BVN theorem says: joint BVN $\Rightarrow$ marginals Normal. The arrow does not reverse. This matters in risk modelling: two assets with Normal marginal returns can have a joint distribution with non-Normal dependence (fat-tailed copula), which changes tail risk completely.
probability / ch18 / q04★★★ Hard

Nifty return $X \sim N(10\%, 20\%^2)$, Bank Nifty return $Y \sim N(12\%, 25\%^2)$, $\rho = 0.7$. A portfolio $Z = 0.6X + 0.4Y$. What is $\text{Var}(Z)$?

A
$0.36(0.04) + 0.16(0.0625) = 0.0244$ — independent assumption
B
$0.36(0.04) + 0.16(0.0625) + 2(0.6)(0.4)(0.7)(0.2)(0.25) = 0.0328$
C
$(0.6+0.4)^2 \times 0.0625 = 0.0625$
D
$0.6^2(0.04) + 0.4^2(0.0625) = 0.0144 + 0.01 = 0.0244$
Answer: B — $0.0328$

$\text{Var}(aX+bY) = a^2\sigma_1^2 + b^2\sigma_2^2 + 2ab\rho\sigma_1\sigma_2$. Here $a=0.6$, $b=0.4$, $\sigma_1=0.2$, $\sigma_2=0.25$, $\rho=0.7$: $$= 0.36(0.04) + 0.16(0.0625) + 2(0.6)(0.4)(0.7)(0.2)(0.25) = 0.0144 + 0.01 + 0.0168 = 0.0328$$ The cross term adds 0.0168 — nearly doubling the independent-case variance. This is why highly correlated assets don't diversify well.
probability / ch18 / q05★★★ Hard

$(X,Y)\sim\text{BVN}(\mu_1=5, \mu_2=8, \sigma_1^2=4, \sigma_2^2=9, \rho=0.5)$. What is $E(X|Y=11)$?

A
$5$ — conditional mean equals unconditional mean
B
$5.5$
C
$6$ — using $E(X|Y=y) = \mu_1 + \rho\frac{\sigma_1}{\sigma_2}(y-\mu_2)$
D
$8$
Answer: C — $6$

$E(X|Y=y) = \mu_1 + \rho\frac{\sigma_1}{\sigma_2}(y-\mu_2) = 5 + 0.5\cdot\frac{2}{3}(11-8) = 5 + 0.5\cdot\frac{2}{3}\cdot 3 = 5 + 1 = 6$. Knowing $Y=11$ (one standard deviation above $\mu_2=8$) shifts our expectation of $X$ upward from 5 to 6, because they are positively correlated. This is the regression formula — the conditional mean is the best linear predictor.
