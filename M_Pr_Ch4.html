Properties of Probability Measures

We have defined the probability space $(\Omega, \mathcal{S}, P)$. Now we squeeze it. Every useful formula in probability theory — inclusion-exclusion, union bounds, Bonferroni inequalities — flows from three axioms. This chapter shows you how.

What We Know Going In

A probability measure $P$ on $(\Omega, \mathcal{S})$ satisfies exactly three axioms:

(i) $P(\Omega) = 1$    (ii) $P(A) \geq 0$ for all $A \in \mathcal{S}$    (iii) $P\!\left(\bigcup_i A_i\right) = \sum_i P(A_i)$ when $A_i$'s are mutually disjoint

Everything else — every theorem, every formula, every inequality in this chapter — is a logical consequence of these three lines. No new assumptions. Just careful reasoning.

Property 1 — Monotonicity

Monotonicity: If $A \subseteq B$, then $P(A) \leq P(B)$.
Bigger set, bigger (or equal) probability. Probability respects containment.
Since $A \subseteq B$, we can write $B$ as a disjoint union: $$B = A \cup (B \setminus A)$$ where $B \setminus A = A^c \cap B$ — the part of $B$ that lies outside $A$. These two sets are disjoint, so by axiom (iii): $$P(B) = P(A) + P(B \setminus A)$$ Since $P(B \setminus A) \geq 0$ by axiom (ii): $$P(B) \geq P(A) \qquad \square$$
Think of it this way. "The Nifty 50 closes above 22,000" is a subset of "the Nifty 50 closes above 20,000." The first event is harder to achieve — it imposes a stricter condition. So its probability should be smaller. Monotonicity formalises this obvious intuition.

Property 2 — The Complement Rule

Complement Rule: For any event $A \subseteq \Omega$: $$P(A^c) = 1 - P(A)$$
Since $A$ and $A^c$ are disjoint and $A \cup A^c = \Omega$: $$P(\Omega) = P(A) + P(A^c)$$ $$1 = P(A) + P(A^c)$$ $$P(A^c) = 1 - P(A) \qquad \square$$ As a bonus: $P(\emptyset) = P(\Omega^c) = 1 - P(\Omega) = 1 - 1 = 0$.

This is one of the most useful tools in probability. Sometimes the complement is far easier to compute than the event itself.

Does $P(A) = 0$ imply $A = \emptyset$? No. This is a trap that catches people every time.

Consider a class where all possible grades are A, B, C, D, F. Suppose every student in the class passed — so getting an F is possible in principle, but no one got one. If we pick a random student, $P(\text{F grade}) = 0$. Yet the event $\{\text{F grade}\}$ is not the empty set — it's a perfectly valid outcome, it just didn't happen to occur in this cohort.

Zero probability $\neq$ impossible. In continuous probability especially, individual outcomes always have zero probability — but they can still occur.

Property 3 — Inclusion-Exclusion for Two Events

When two events overlap, we can't just add their probabilities — we'd count the overlap twice. The fix is to subtract it once.

For any two events $A, B \in \mathcal{S}$: $$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$
Partition $A \cup B$ into three disjoint pieces using elements $x \in A \setminus B$, $y \in B \setminus A$, and $z \in A \cap B$: $$A \cup B = (A \setminus B) \cup (B \setminus A) \cup (A \cap B)$$ Also note that $A = (A \setminus B) \cup (A \cap B)$, so $P(A) = P(A \setminus B) + P(A \cap B)$, giving $P(A \setminus B) = P(A) - P(A \cap B)$. Similarly $P(B \setminus A) = P(B) - P(A \cap B)$. Therefore: $$P(A \cup B) = P(A \setminus B) + P(B \setminus A) + P(A \cap B)$$ $$= [P(A) - P(A \cap B)] + [P(B) - P(A \cap B)] + P(A \cap B)$$ $$= P(A) + P(B) - P(A \cap B) \qquad \square$$
🎲 Worked Example — Die Roll

Throw a fair die. $\Omega = \{1,2,3,4,5,6\}$.

Let $A = \{\text{outcome} \leq 3\} = \{1,2,3\}$, so $P(A) = \frac{3}{6} = \frac{1}{2}$.

Let $B = \{\text{outcome is even}\} = \{2,4,6\}$, so $P(B) = \frac{3}{6} = \frac{1}{2}$.

What is $P(A \cup B)$ — the probability of getting $\leq 3$ or an even number?

$A \cup B = \{1,2,3,4,6\}$ — five favourable outcomes.

$A \cap B = \{2\}$ — the outcome that is both $\leq 3$ and even. So $P(A \cap B) = \frac{1}{6}$.

$$P(A \cup B) = \frac{1}{2} + \frac{1}{2} - \frac{1}{6} = \frac{5}{6}$$

Verify directly: $\frac{5}{6}$. ✓

Property 4 — Sub-additivity (Union Bound)

Boole's Inequality / Union Bound: For any events $A_1, A_2, \ldots, A_n$: $$P\!\left(\bigcup_{i=1}^n A_i\right) \leq \sum_{i=1}^n P(A_i)$$

This follows immediately from inclusion-exclusion — we subtract $P(A \cap B)$ which is non-negative, so adding probabilities only overestimates the union. The union bound is crude but fast — and in many financial applications, a quick overestimate is exactly what you need.

The union bound is everywhere in risk management. If there's a 2% chance of a credit default on bond A and a 3% chance on bond B, the probability of at least one default is at most $2\% + 3\% = 5\%$. This overestimates (since defaults can be correlated), but it's a safe upper bound — and safe is what risk managers want when sizing reserves.

Property 5 — Inclusion-Exclusion for Three Events

$$P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(A \cap B) - P(A \cap C) - P(B \cap C) + P(A \cap B \cap C)$$
Treat $A \cup B \cup C = A \cup (B \cup C)$ and apply the two-event formula twice: $$P(A \cup B \cup C) = P(A) + P(B \cup C) - P(A \cap (B \cup C))$$ Expand $P(B \cup C) = P(B) + P(C) - P(B \cap C)$. For the last term, use distributivity: $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$, so: $$P(A \cap (B \cup C)) = P(A \cap B) + P(A \cap C) - P(A \cap B \cap C)$$ Substitute everything in and simplify — the formula above drops out. $\square$

The pattern: add individual probabilities, subtract pairwise intersections, add triple intersections, subtract quadruple intersections, and so on. This is the Inclusion-Exclusion Principle — it generalises to $n$ events.

General Inclusion-Exclusion: $$P\!\left(\bigcup_{i=1}^n A_i\right) = \sum_{i=1}^n P(A_i) - \sum_{i < j} P(A_i \cap A_j) + \sum_{i < j < k} P(A_i \cap A_j \cap A_k) - \cdots + (-1)^{n+1} P(A_1 \cap \cdots \cap A_n)$$ Proved by induction on $n$.

Property 6 — Bonferroni's Inequality

Inclusion-exclusion gives the exact probability of a union. Bonferroni gives a lower bound on the probability of an intersection — extremely useful when you know individual probabilities but not the joint structure.

Bonferroni's Inequality: For any events $A_1, A_2, \ldots, A_n$: $$P\!\left(\bigcap_{i=1}^n A_i\right) \geq 1 - \sum_{i=1}^n P(A_i^c)$$
Base case ($n=2$): We know $P(A \cup B) \leq P(A) + P(B)$. Replace $A$ with $A^c$ and $B$ with $B^c$: $$P(A^c \cup B^c) \leq P(A^c) + P(B^c)$$ So $1 - P(A^c \cup B^c) \geq 1 - P(A^c) - P(B^c)$. By De Morgan, $(A^c \cup B^c)^c = A \cap B$, giving: $$P(A \cap B) \geq 1 - P(A^c) - P(B^c) \checkmark$$ Inductive step: Assume it holds for $m$ events. For $m+1$ events, treat $(A_1 \cap \cdots \cap A_m)$ as one event and $A_{m+1}$ as another. Apply the base case, then substitute the inductive hypothesis for $P(A_1 \cap \cdots \cap A_m)$. The result follows. $\square$
Bonferroni says: if you want all $n$ events to happen simultaneously, and each individual event is very likely, then their joint occurrence is also likely — as long as the individual failure probabilities $P(A_i^c)$ are small in aggregate. If each of 100 events has a 99.9% chance of occurring, Bonferroni guarantees their simultaneous occurrence has probability at least $1 - 100 \times 0.001 = 90\%$. This is the logic behind stress-testing portfolios under multiple simultaneous conditions.

All Properties at a Glance

① Monotonicity
Bigger set, bigger probability
$A \subseteq B \Rightarrow P(A) \leq P(B)$
② Complement
What doesn't happen
$P(A^c) = 1 - P(A)$,   $P(\emptyset) = 0$
③ Inclusion-Exclusion
Exact union probability
$P(A \cup B) = P(A) + P(B) - P(A \cap B)$
④ Union Bound
Fast upper bound on union
$P\!\left(\bigcup A_i\right) \leq \sum P(A_i)$
⑤ Inclusion-Exclusion ($n$)
General union formula
Alternating sum of all intersection probabilities
⑥ Bonferroni
Lower bound on intersection
$P\!\left(\bigcap A_i\right) \geq 1 - \sum P(A_i^c)$

probability / ch04 / q01 ★ Easy

If $P(A) = 0$, which of the following must be true?

A
$A = \emptyset$
B
$A$ cannot occur in any trial
C
$P(A^c) = 1$, but $A$ may still be a non-empty set
D
$A$ is not a valid event in $\mathcal{S}$
Answer: C

$P(A) = 0$ and $A = \emptyset$ are not the same thing. By the complement rule, $P(A^c) = 1 - 0 = 1$ — this part is always true. But $A$ can be a non-empty set with probability zero. Example: in a class where all students passed, the event "student gets an F" is non-empty (F is a valid grade) but has probability zero because no one got it. Zero probability ≠ impossible. This distinction becomes especially important in continuous probability, where every individual outcome has zero probability.
probability / ch04 / q02 ★★ Medium

A fair die is rolled. Let $A$ = {outcome is prime} = {2,3,5} and $B$ = {outcome is odd} = {1,3,5}. What is $P(A \cup B)$?

A
$\dfrac{2}{3}$
B
$\dfrac{2}{3}$... wait — $\dfrac{4}{6} = \dfrac{2}{3}$
C
$1$
D
$\dfrac{5}{6}$
Answer: B — $\dfrac{2}{3}$

$P(A) = \frac{3}{6} = \frac{1}{2}$ (primes: 2,3,5). $P(B) = \frac{3}{6} = \frac{1}{2}$ (odds: 1,3,5).

$A \cap B = \{3, 5\}$ — numbers that are both prime and odd. $P(A \cap B) = \frac{2}{6} = \frac{1}{3}$.

$$P(A \cup B) = \frac{1}{2} + \frac{1}{2} - \frac{1}{3} = 1 - \frac{1}{3} = \frac{2}{3}$$ Verify: $A \cup B = \{1,2,3,5\}$ — four outcomes. $\frac{4}{6} = \frac{2}{3}$. ✓
probability / ch04 / q03 ★★ Medium

$P(A) = 0.6$ and $P(B) = 0.5$. Using the union bound, what can you say about $P(A \cup B)$?

A
$P(A \cup B) = 1.1$ — this is impossible, so the events must be dependent
B
$P(A \cup B) = 1.1$
C
$P(A \cup B) \leq 1$ — the union bound gives an upper bound of $\min(1.1, 1) = 1$
D
$P(A \cup B) = 0.55$
Answer: C

The union bound says $P(A \cup B) \leq P(A) + P(B) = 0.6 + 0.5 = 1.1$. But probability is bounded above by 1, so the effective upper bound is $\min(1.1, 1) = 1$. The union bound simply tells us $P(A \cup B) \leq 1$ here — uninformative but not wrong.

The union bound is most useful when $P(A) + P(B)$ is well below 1. When it exceeds 1, the bound is trivially saturated. The union bound is a tool — know when it's tight and when it isn't.
probability / ch04 / q04 ★★★ Hard

An options desk runs 3 independent strategies. Each strategy has a 95% probability of not losing money on a given day. Using Bonferroni's inequality, what is the lower bound on the probability that all three strategies are profitable simultaneously?

A
$0.95^3 \approx 85.7\%$
B
$1 - 3 \times 0.05 = 85\%$
C
$3 \times 0.95 = 285\%$ — not valid
D
$0.95 / 3 \approx 31.7\%$
Answer: B — $85\%$

Let $A_i$ = "strategy $i$ is profitable." Then $P(A_i) = 0.95$, so $P(A_i^c) = 0.05$ for each $i$.

Bonferroni's inequality states: $$P(A_1 \cap A_2 \cap A_3) \geq 1 - P(A_1^c) - P(A_2^c) - P(A_3^c) = 1 - 0.05 - 0.05 - 0.05 = 0.85$$ So all three strategies are simultaneously profitable with probability at least 85%.

Note: the exact answer (since they're independent) is $0.95^3 \approx 85.7\%$. Bonferroni gives 85% — slightly conservative, but it works without needing to know whether the strategies are independent. That's its power.
