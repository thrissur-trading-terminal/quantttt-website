Order Statistics

Given $n$ iid observations, sort them. The resulting sorted values $X_{(1)}\leq X_{(2)}\leq\cdots\leq X_{(n)}$ are the order statistics. This chapter derives the PDF of the $r$th order statistic using a beautiful telescoping argument, then computes moments of the median via the Beta distribution.

Definition

Let $X_1, X_2, \ldots, X_n$ be iid with CDF $F(x)$ and PDF $f(x)$. Arrange the $n$ values in increasing order: $$X_{(1)} \leq X_{(2)} \leq \cdots \leq X_{(r)} \leq \cdots \leq X_{(n)}$$ These are the order statistics. $X_{(r)}$ is the $r$th order statistic — the $r$th smallest value.

$X_{(1)} = \min(X_1,\ldots,X_n)$, $\quad X_{(n)} = \max(X_1,\ldots,X_n)$, $\quad X_{(\lceil n/2\rceil)}$ = sample median.
Each $X_{(r)}$ is a genuine random variable. For $n=3$ with values $\{1.5, 2.75, 4\}$: all $3! = 6$ orderings give the same order statistics $X_{(1)}=1.5$, $X_{(2)}=2.75$, $X_{(3)}=4$. The order statistics summarise the sample by stripping away all information about which label went where.

PDF of the Maximum and Minimum

Maximum $X_{(n)}$: $$F_{(n)}(x) = P(X_{(n)}\leq x) = P(\text{all }X_i\leq x) = [F(x)]^n$$ $$f_{(n)}(x) = n[F(x)]^{n-1}f(x)$$ Minimum $X_{(1)}$: $$F_{(1)}(x) = P(\text{at least one }X_i\leq x) = 1-P(\text{all }X_i>x) = 1-(1-F(x))^n$$ $$f_{(1)}(x) = n(1-F(x))^{n-1}f(x)$$
Verification: $X_i\sim\text{Exp}(\lambda)$, find PDF of $X_{(1)}$

$f(x) = \lambda e^{-\lambda x}$, $F(x) = 1-e^{-\lambda x}$. Then $1-F(x) = e^{-\lambda x}$:

$$f_{(1)}(x) = n(e^{-\lambda x})^{n-1}\lambda e^{-\lambda x} = n\lambda\, e^{-n\lambda x} = \text{Exp}(n\lambda) \checkmark$$

The minimum of $n$ independent Exp$(\lambda)$ variables is Exp$(n\lambda)$.

PDF of the $r$th Order Statistic

$$\boxed{f_{(r)}(x) = \frac{n!}{(r-1)!(n-r)!}[F(x)]^{r-1}[1-F(x)]^{n-r}f(x) = \frac{1}{B(r,\,n-r+1)}[F(x)]^{r-1}[1-F(x)]^{n-r}f(x)}$$
Via CDF differentiation: $P(X_{(r)}\leq x) = P(\text{at least }r\text{ of }X_1,\ldots,X_n\leq x)$. By the Binomial argument ($F(x)$ = probability any single $X_i\leq x$): $$F_{(r)}(x) = \sum_{k=r}^n\binom{n}{k}[F(x)]^k[1-F(x)]^{n-k}$$ Differentiate term by term. Each term $\frac{d}{dx}\binom{n}{k}F^k(1-F)^{n-k}$ gives a telescoping pair: $$= \frac{n!}{(k-1)!(n-k)!}F^{k-1}(1-F)^{n-k}f - \frac{n!}{k!(n-k-1)!}F^k(1-F)^{n-k-1}f$$ Summing from $k=r$ to $k=n$: consecutive negative and positive terms cancel. The last term at $k=n$ has $(-1)!$ which doesn't exist, so it's discarded. Only the first positive term from $k=r$ survives: $$f_{(r)}(x) = \frac{n!}{(r-1)!(n-r)!}[F(x)]^{r-1}[1-F(x)]^{n-r}f(x) \qquad \square$$
Intuitive derivation: Think of $f_{(r)}(x)\Delta x$ as the probability that $X_{(r)}$ falls in $(x, x+\Delta x)$. This requires:
  — exactly $r-1$ of the $n$ variables below $x$: probability $\binom{n}{r-1}[F(x)]^{r-1}$
  — one of the remaining $n-r+1$ falls in $(x, x+\Delta x)$: probability $\binom{n-r+1}{1}f(x)\Delta x$
  — the remaining $n-r$ fall above $x+\Delta x$: probability $[1-F(x)]^{n-r}$

Multiplying and simplifying: $f_{(r)}(x) = \frac{n!}{(r-1)!(n-r)!}F^{r-1}(1-F)^{n-r}f(x)$. Same formula, much easier to remember.
Verify for $r=1$: reduces to minimum formula
$$\frac{1}{B(1,n)}[F(x)]^0[1-F(x)]^{n-1}f(x) = \frac{\Gamma(n+1)}{\Gamma(1)\Gamma(n)}[1-F(x)]^{n-1}f(x) = n[1-F(x)]^{n-1}f(x) \checkmark$$

Worked Example: Median of $U(0,1)$ Sample

$n=5$, $X_i\sim U(0,1)$. Find $E(X_{(3)})$ and $\text{Var}(X_{(3)})$.

$F(x)=x$, $f(x)=1$ on $[0,1]$.

$$f_{(3)}(x) = \frac{1}{B(3,3)}x^2(1-x)^2, \quad 0\leq x\leq 1 \implies X_{(3)}\sim\beta_1(3,3)$$

Mean: $E(X_{(3)}) = \frac{3}{3+3} = \frac{1}{2}$. ✓ The median of a symmetric distribution is the mean.

Second moment:

$$E(X_{(3)}^2) = \frac{1}{B(3,3)}\int_0^1 x^4(1-x)^2\,dx = \frac{B(5,3)}{B(3,3)} = \frac{\Gamma(6)/(\Gamma(5)\Gamma(3))\cdot\Gamma(5)\Gamma(3)/\Gamma(8)}{\Gamma(6)/(\Gamma(3)\Gamma(3))} = \frac{B(5,3)}{B(3,3)}$$ $$= \frac{\Gamma(5)\Gamma(3)/\Gamma(8)}{\Gamma(3)\Gamma(3)/\Gamma(6)} = \frac{4!\cdot 2!}{7!}\cdot\frac{5!}{2!\cdot 2!} = \frac{12}{42} = \frac{2}{7}$$ $$\text{Var}(X_{(3)}) = \frac{2}{7} - \left(\frac{1}{2}\right)^2 = \frac{2}{7}-\frac{1}{4} = \frac{8-7}{28} = \frac{1}{28}$$
Order statistics are fundamental to risk management. In NSE options, daily settlement prices use trimmed means that eliminate outliers — explicitly discarding the $X_{(1)}$ and $X_{(n)}$ order statistics. The VaR at 99% confidence is the 1st percentile of the return distribution — the first order statistic of a 100-observation window. Every percentile, every quantile, every quartile in finance is an order statistic.

$X_1,\ldots,X_n$ iid with CDF $F(x)$. What is the CDF of $X_{(n)} = \max(X_1,\ldots,X_n)$?

A
$1-(1-F(x))^n$
B
$[F(x)]^n$
C
$nF(x)$
D
$1-[F(x)]^n$
Answer: B — $[F(x)]^n$

$P(\max\leq x) = P(\text{all }X_i\leq x) = [F(x)]^n$ by independence. Option A is the CDF of the minimum $X_{(1)}$. Option D is $1-[F(x)]^n = P(\min\leq x)$, another way to write the min's CDF.
probability / ch22 / q02★★ Medium

$X_1,\ldots,X_5$ iid $U(0,1)$. What is the distribution of $X_{(3)}$ (the median)?

A
$U(0,1)$ — each $X_i$ is uniform, so the median is too
B
$\beta_1(3,2)$
C
$\beta_1(3,3)$
D
$N(0.5, 1/28)$
Answer: C — $\beta_1(3,3)$

$f_{(r)}(x) = \frac{1}{B(r,n-r+1)}[F(x)]^{r-1}[1-F(x)]^{n-r}f(x)$. With $r=3$, $n=5$, $F(x)=x$, $f(x)=1$: $f_{(3)}(x) = \frac{1}{B(3,3)}x^2(1-x)^2 = \beta_1(3,3)$. Note: $n-r+1 = 5-3+1=3$. The median has $r-1=2$ below and $n-r=2$ above, giving equal Beta parameters — symmetric distribution with mean $1/2$.

$n=4$ iid Exp$(2)$ variables. The PDF of the minimum $X_{(1)}$ is:

A
$2e^{-2x}$ — same as each individual variable
B
$8e^{-2x}$
C
$8e^{-8x}$ — Exp$(n\lambda) = $ Exp$(8)$
D
$4e^{-4x}$
Answer: C — $8e^{-8x}$

Min of $n$ iid Exp$(\lambda)$ is Exp$(n\lambda)$. Here $n=4$, $\lambda=2$: min $\sim$ Exp$(8)$, PDF $= 8e^{-8x}$. Using the formula: $f_{(1)}(x) = n(1-F(x))^{n-1}f(x) = 4(e^{-2x})^3\cdot 2e^{-2x} = 8e^{-8x}$. ✓
probability / ch22 / q04★★★ Hard

$X_1,\ldots,X_7$ iid $U(0,1)$. What is $E(X_{(4)})$, the expected value of the median?

A
$4/7$ — 4th out of 7, so fraction $4/7$
B
$1/2$ — median of symmetric distribution
C
$4/8 = 1/2$ — same thing, different reasoning
D
$3/8$ — using $r/(n+1)$ formula incorrectly
Answer: B — $\frac{1}{2}$

$X_{(4)}\sim\beta_1(4,4)$ with $r=4$, $n=7$, $n-r+1=4$. $E(\beta_1(\alpha,\beta)) = \alpha/(\alpha+\beta) = 4/8 = 1/2$. More generally for $U(0,1)$: $E(X_{(r)}) = r/(n+1)$. Here $4/8 = 1/2$. The median of $n$ uniform samples is always $1/2$ for odd $n$.
probability / ch22 / q05★★★ Hard

The PDF of $X_{(r)}$ can be remembered via the intuitive argument: (r-1) below $x$, one at $x$, (n-r) above. Which formula correctly captures this?

A
$\binom{n}{r}[F(x)]^r[1-F(x)]^{n-r}f(x)$
B
$\binom{n}{r-1}\cdot\binom{n-r+1}{1}[F(x)]^{r-1}[1-F(x)]^{n-r}f(x)$
C
$n[F(x)]^{r-1}[1-F(x)]^{n-r}f(x)$
D
$\frac{n!}{r!(n-r)!}[F(x)]^r[1-F(x)]^{n-r}f(x)$
Answer: B

Choose $r-1$ from $n$ to be below $x$: $\binom{n}{r-1}[F(x)]^{r-1}$. Then choose 1 from the remaining $n-r+1$ to fall in $(x, x+dx)$: $\binom{n-r+1}{1}f(x)dx$. The remaining $n-r$ fall above $x$: $[1-F(x)]^{n-r}$. Multiplying and simplifying: $\frac{n!}{(r-1)!(n-r)!}F^{r-1}(1-F)^{n-r}f(x)$. This intuitive argument gives exactly the right formula without any calculus.
