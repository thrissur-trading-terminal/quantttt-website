Almost Sure Convergence & The Hierarchy

This chapter proves rigorously that convergence in probability implies convergence in distribution, identifies the special case where the reverse holds (convergence to a constant), introduces the strongest classical mode — almost sure convergence — and establishes the complete three-tier hierarchy that governs all subsequent limit theorems.

Proof: Convergence in Probability $\Rightarrow$ Convergence in Distribution

Claim: $X_n\xrightarrow{P}X \Rightarrow X_n\xrightarrow{d}X$.

Fix $x\in\mathbb{R}$ and $\varepsilon>0$. Decompose $\{X_n\leq x\}$: $$\{X_n\leq x\} = \underbrace{\{X_n\leq x\}\cap\{X\geq x+\varepsilon\}}_{\subseteq\{|X_n-X|\geq\varepsilon\}} \cup \{X_n\leq x\}\cap\{X < x+\varepsilon\}$$ The second set is contained in $\{X < x+\varepsilon\}$, so: $$F_n(x) = P(X_n\leq x) \leq P(|X_n-X|\geq\varepsilon) + F(x+\varepsilon) \qquad \cdots (1)$$ Similarly, $P(X\leq x-\varepsilon) = P(\{X\leq x-\varepsilon\}\cap\{X_n>x\}) + P(\{X\leq x-\varepsilon\}\cap\{X_n\leq x\})$. The first term is $\leq P(|X_n-X|\geq\varepsilon)$; the second is $\leq F_n(x)$: $$F(x-\varepsilon) - P(|X_n-X|\geq\varepsilon) \leq F_n(x) \qquad \cdots (2)$$ Combining (1) and (2): $$F(x-\varepsilon) - P(|X_n-X|\geq\varepsilon) \leq F_n(x) \leq F(x+\varepsilon) + P(|X_n-X|\geq\varepsilon)$$ Since $X_n\xrightarrow{P}X$: $P(|X_n-X|\geq\varepsilon)\to 0$. Taking $n\to\infty$: $$F(x-\varepsilon) \leq \liminf_n F_n(x) \leq \limsup_n F_n(x) \leq F(x+\varepsilon)$$ Since $x$ is a continuity point of $F$: as $\varepsilon\to 0$, $F(x\pm\varepsilon)\to F(x)$. Therefore $F_n(x)\to F(x)$. $\square$

When Does $\xrightarrow{d}$ Imply $\xrightarrow{P}$?

If $X_n\xrightarrow{d}k$ (convergence to a constant), then $X_n\xrightarrow{P}k$.
The degenerate CDF at $k$ is $F(x) = \mathbf{1}_{x\geq k}$. Since $X_n\xrightarrow{d}k$: for any $\delta>0$, $F_n(k-\delta)\to 0$ and $F_n(k+\delta)\to 1$. Therefore: $$P(|X_n-k|<\delta) = P(k-\delta < X_n \leq k+\delta) = F_n(k+\delta) - F_n(k-\delta) \to 1-0 = 1$$ So $P(|X_n-k|\geq\varepsilon)\to 0$ for every $\varepsilon>0$. $\square$
Illustration: $X_n$ takes $k\pm 1/n$ with prob $1/2$ each

Each $F_n$ is a step function with jumps at $k-1/n$ and $k+1/n$. As $n\to\infty$, both jump points squeeze toward $k$, so $F_n\to$ the degenerate CDF at $k$. Therefore $X_n\xrightarrow{d}k$, and by the theorem, $X_n\xrightarrow{P}k$.

Worked Example: Max of $U(-1,1)$ Converges in Probability

$X_1,\ldots,X_n$ iid $U(-1,1)$. Show $Y_n=\max(X_1,\ldots,X_n)\xrightarrow{P}1$.

For any $\varepsilon>0$:

$$P(|Y_n-1|\geq\varepsilon) = P(Y_n\geq 1+\varepsilon) + P(Y_n\leq 1-\varepsilon)$$

Since $X_i\leq 1$ always: $P(Y_n\geq 1+\varepsilon) = 0$.

For the second term: $F_{U(-1,1)}(x) = (x+1)/2$, so $P(X\leq 1-\varepsilon) = (1-\varepsilon/2) = 1-\varepsilon/2$. By independence:

$$P(Y_n\leq 1-\varepsilon) = \left(1-\frac{\varepsilon}{2}\right)^n \to 0 \text{ as } n\to\infty$$

Therefore $P(|Y_n-1|\geq\varepsilon)\to 0$: $Y_n\xrightarrow{P}1$. And since convergence in probability implies convergence in distribution: $Y_n\xrightarrow{d}1$. ✓

Almost Sure Convergence

$\{X_n\}$ converges almost surely to $X$, written $X_n\xrightarrow{a.s.}X$, if: $$P\!\left(\{\omega\in\Omega : X_n(\omega)\to X(\omega)\text{ as }n\to\infty\}\right) = 1$$ The set of $\omega$ where the pointwise convergence fails may be non-empty, but it must have probability zero.
For ordinary (real-number) sequences: either $x_n\to x$ or it doesn't. For almost sure convergence: we allow failure on a "probability-zero" set of sample paths. This is a much stronger statement than convergence in probability, which only requires that the probability of being far from $X$ goes to zero — it doesn't require actual convergence for individual $\omega$.
Example 1: Unit Square

$\Omega = [0,1]\times[0,1]$, $\omega=(x,y)$. Define:

$X_n(\omega) = \begin{cases}1 & \text{if }x\neq y \\ 0 & \text{if }x=y\end{cases}$, and $X(\omega)=1$ for all $\omega$.

The set $A = \{(x,y): x\neq y\}$ has $P(A) = 1$ (the diagonal $\{x=y\}$ has measure zero). On $A$: $X_n(\omega)=1=X(\omega)$ for all $n$, so $X_n(\omega)\to X(\omega)$. Therefore $P(\{\omega:X_n(\omega)\to X(\omega)\})\geq P(A) = 1$. So $X_n\xrightarrow{a.s.}X$.

Example 2: Deterministic sequence

$X_n(\omega) = 2 + 1/n$ for all $\omega\in\Omega$. For every $\omega$: $X_n(\omega) = 2+1/n\to 2$. So $P(\{\omega: X_n(\omega)\to 2\}) = P(\Omega) = 1$. Therefore $X_n\xrightarrow{a.s.}2$.

The Complete Hierarchy

$$X_n\xrightarrow{a.s.}X \;\Rightarrow\; X_n\xrightarrow{P}X \;\Rightarrow\; X_n\xrightarrow{d}X$$ None of these arrows reverses in general. The almost sure convergence is the strongest classical mode; convergence in distribution is the weakest.

Special case: If $X\equiv k$ (constant), then $\xrightarrow{d}k$ $\Rightarrow$ $\xrightarrow{P}k$ (the second arrow reverses).
Think of the three modes as measuring "how close" $X_n$ is to $X$:

a.s.: almost every sample path of $X_n$ eventually tracks $X$ pointwise.
Probability: the probability of being far from $X$ vanishes, but individual paths may oscillate.
Distribution: only the laws match up; individual paths can be arbitrarily far apart (the Bernoulli counterexample from Ch25).

The Laws of Large Numbers (next chapter) will use these: the Weak LLN uses convergence in probability; the Strong LLN uses almost sure convergence.
In algorithmic trading backtests: the Weak LLN guarantees that the sample Sharpe ratio converges in probability to the true Sharpe as the number of trades grows — enough for practical purposes. The Strong LLN gives the stronger guarantee that the sample paths themselves converge with probability 1 — the theoretical foundation for why long backtests reliably estimate true performance. Understanding which mode applies tells you exactly how much confidence to place in empirical estimates.

Which statement correctly describes the hierarchy of modes of convergence?

A
$\xrightarrow{d}$ $\Rightarrow$ $\xrightarrow{P}$ $\Rightarrow$ $\xrightarrow{a.s.}$
B
$\xrightarrow{a.s.}$ $\Rightarrow$ $\xrightarrow{P}$ $\Rightarrow$ $\xrightarrow{d}$

All three modes are equivalent

$\xrightarrow{P}$ $\Rightarrow$ $\xrightarrow{a.s.}$ $\Rightarrow$ $\xrightarrow{d}$
Answer: B

Almost sure is the strongest: $\xrightarrow{a.s.}\Rightarrow\xrightarrow{P}\Rightarrow\xrightarrow{d}$. None of these reverses in general. The Weak LLN uses $\xrightarrow{P}$; the Strong LLN uses $\xrightarrow{a.s.}$.

$X_1,\ldots,X_n$ iid $U(0,2)$. Does $Y_n = \max(X_1,\ldots,X_n)$ converge in probability, and if so, to what?

A
No convergence — the max keeps growing
B
$Y_n\xrightarrow{P}2$ — $P(Y_n\leq 2-\varepsilon)=(1-\varepsilon/2)^n\to 0$
C
$Y_n\xrightarrow{P}1$ — converges to the mean
D
$Y_n\xrightarrow{d}U(0,2)$ — same distribution as each $X_i$
Answer: B — $Y_n\xrightarrow{P}2$

$P(|Y_n-2|\geq\varepsilon) = P(Y_n\leq 2-\varepsilon) = ((2-\varepsilon)/2)^n = (1-\varepsilon/2)^n\to 0$. The max of $n$ uniform variables concentrates at the upper bound as $n\to\infty$.
probability / ch26 / q03★★ Medium

$X_n(\omega) = \omega + 1/n$ for all $\omega\in[0,1]$. Does $X_n$ converge almost surely, and to what?

A
$X_n\xrightarrow{a.s.}X$ where $X(\omega)=\omega$ — every $\omega$ satisfies $X_n(\omega)\to\omega$
B
No a.s. convergence — $1/n$ doesn't go to zero
C
$X_n\xrightarrow{a.s.}1$ — the $1/n$ term dominates for large $n$
D
$X_n\xrightarrow{P}\omega$ but not a.s.
Answer: A — $X_n\xrightarrow{a.s.}\omega$

For every $\omega\in[0,1]$: $X_n(\omega)=\omega+1/n\to\omega$ as $n\to\infty$. So $P(\{\omega: X_n(\omega)\to\omega\}) = P([0,1]) = 1$. Almost sure convergence is the strongest mode — here it holds at every single $\omega$ (not just with probability 1 but with probability 1 trivially because it holds everywhere).
probability / ch26 / q04★★★ Hard

In the sandwich proof, the key step is: $\{X_n\leq x\}\cap\{X\geq x+\varepsilon\}$ is contained in $\{|X_n-X|\geq\varepsilon\}$. Why?

A
Because $X_n$ and $X$ are identically distributed
B
If $X_n\leq x$ and $X\geq x+\varepsilon$, then $X-X_n\geq\varepsilon$, so $|X_n-X|\geq\varepsilon$
C
Because $F_n(x)\leq F(x+\varepsilon)$ always holds for CDFs
D
Because $P(X_n\leq x)\leq 1$ trivially
Answer: B

If $X_n(\omega)\leq x$ and $X(\omega)\geq x+\varepsilon$, then $X(\omega)-X_n(\omega)\geq\varepsilon$, so $|X_n(\omega)-X(\omega)|\geq\varepsilon$. This set containment is what lets us bound $P(X_n\leq x, X\geq x+\varepsilon)\leq P(|X_n-X|\geq\varepsilon)$, which vanishes when $X_n\xrightarrow{P}X$. The sandwich then squeezes $F_n(x)$ to $F(x)$.
probability / ch26 / q05★★★ Hard

$X_n\xrightarrow{d}5$ (converges in distribution to the constant 5). Does $X_n\xrightarrow{P}5$?

A
No — convergence in distribution never implies convergence in probability
B
Yes — when the limit is a constant, $\xrightarrow{d}$ and $\xrightarrow{P}$ are equivalent
C
Only if $X_n$ are identically distributed
D
Only if $X_n\geq 5$ almost surely
Answer: B — Yes, convergence in probability follows

Theorem: $X_n\xrightarrow{d}k$ $\Rightarrow$ $X_n\xrightarrow{P}k$. Proof: the degenerate CDF gives $F_n(5-\delta)\to 0$ and $F_n(5+\delta)\to 1$, so $P(|X_n-5|<\delta)\to 1$. This is the one case where convergence in distribution is as strong as convergence in probability. It fails when the limit is a non-degenerate RV because then the joint structure of $(X_n,X)$ can conspire against probability convergence.
