Variance

Two distributions can share the same mean and be completely different. Variance measures how spread out a distribution is around its centre. Together, mean and variance are the two numbers that most concisely summarise a distribution — and they are the parameters $\mu$ and $\sigma^2$ you've seen throughout this course.

Expectation of a Function — No Need to Re-derive the Distribution

If $Y = g(X)$, you don't need to find the full distribution of $Y$ to compute $E(Y)$. You can compute directly:

Law of the Unconscious Statistician (LOTUS): $$E[g(X)] = \sum_x g(x)\,p_x \quad \text{(discrete)} \qquad E[g(X)] = \int_{-\infty}^\infty g(x)\,f(x)\,dx \quad \text{(continuous)}$$
Example — $E(X^2)$ for $N(0,1)$

$E(X^2) = \int_{-\infty}^\infty x^2 \frac{1}{\sqrt{2\pi}}e^{-x^2/2}dx$. Since $x^2 e^{-x^2/2}$ is even:

$$= \frac{2}{\sqrt{2\pi}}\int_0^\infty x^2 e^{-x^2/2}dx$$

Substitute $z = x^2$, $dx = \frac{dz}{2\sqrt{z}}$:

$$= \frac{1}{\sqrt{2\pi}}\int_0^\infty z^{3/2-1}e^{-z/2}dz = \frac{1}{\sqrt{2\pi}}\cdot\frac{\Gamma(3/2)}{(1/2)^{3/2}} = \frac{1}{\sqrt{2\pi}}\cdot\frac{\frac{1}{2}\sqrt{\pi}}{\frac{1}{2\sqrt{2}}} = 1$$

Shortcut: $X \sim N(0,1) \Rightarrow X^2 \sim \chi^2_{(1)} = \Gamma(\frac{1}{2},\frac{1}{2})$, so $E(X^2) = \frac{\alpha}{\lambda} = \frac{1/2}{1/2} = 1$. Same answer, less work.

Why Expectation Alone Isn't Enough

$\text{Bin}(100, \tfrac{1}{4})$ has $E(X) = 25$ and takes values in $\{0,1,\ldots,100\}$.

$\text{Bin}(150, \tfrac{1}{6})$ has $E(X) = 25$ and takes values in $\{0,1,\ldots,150\}$.

Same mean. Completely different distributions. We need a second descriptor.

Definition of Variance

The variance of a random variable $X$ with $E(X) = \mu$ is: $$\text{Var}(X) = E[(X - \mu)^2] = \begin{cases}\displaystyle\sum_x (x-\mu)^2\,p_x & \text{discrete} \\ \displaystyle\int_{-\infty}^\infty (x-\mu)^2 f(x)\,dx & \text{continuous}\end{cases}$$ It measures the expected squared deviation from the mean. The standard deviation is $\sigma = +\sqrt{\text{Var}(X)}$.

Computational Formula

$$\text{Var}(X) = E(X^2) - (E(X))^2$$ Almost always easier to use than the definition directly.
$$\text{Var}(X) = \sum_x (x - \mu)^2 p_x = \sum_x (x^2 - 2x\mu + \mu^2)\,p_x$$ $$= \sum_x x^2 p_x - 2\mu\sum_x x\,p_x + \mu^2\sum_x p_x = E(X^2) - 2\mu\cdot\mu + \mu^2\cdot 1 = E(X^2) - \mu^2 \qquad \square$$ Note: $\mu = E(X)$ is a constant (a fixed number), so it comes outside the sum.

Variances of Named Distributions

Bernoulli($p$)
$\text{Var}(X) = pq$

$E(X^2) = 1^2 \cdot p + 0^2 \cdot q = p$. So $\text{Var}(X) = p - p^2 = p(1-p) = pq$.

Maximum variance at $p = \frac{1}{2}$ (fair coin): $\text{Var} = \frac{1}{4}$. Zero variance at $p=0$ or $p=1$ (deterministic outcomes).

Binomial($n,p$)
$\text{Var}(X) = npq$

Write $x^2 = x(x-1) + x$. Then $E(X^2) = S + E(X)$ where $S = \sum x(x-1)\binom{n}{x}p^x q^{n-x}$.

For $x(x-1)\binom{n}{x} = n(n-1)\binom{n-2}{x-2}$, substitute $j=x-2$:

$$S = n(n-1)p^2\sum_{j=0}^{n-2}\binom{n-2}{j}p^j q^{n-2-j} = n(n-1)p^2$$ $$E(X^2) = n(n-1)p^2 + np \implies \text{Var}(X) = n(n-1)p^2 + np - (np)^2 = np(1-p) = npq$$
Poisson($\lambda$)
$\text{Var}(X) = \lambda$

Same $x(x-1)+x$ trick: $E(X^2) = \lambda^2 + \lambda$. Therefore $\text{Var}(X) = \lambda^2 + \lambda - \lambda^2 = \lambda$.

For Poisson: mean = variance = $\lambda$. This equality is diagnostic — if observed variance $\gg$ mean in count data, it suggests overdispersion and a Poisson model is inappropriate.

Exp($\lambda$)
$\text{Var}(X) = \dfrac{1}{\lambda^2}$

$E(X^2) = \int_0^\infty x^2 \lambda e^{-\lambda x}dx$. Substitute $z = \lambda x$:

$$= \frac{1}{\lambda^2}\int_0^\infty z^2 e^{-z}dz = \frac{\Gamma(3)}{\lambda^2} = \frac{2}{\lambda^2}$$ $$\text{Var}(X) = \frac{2}{\lambda^2} - \frac{1}{\lambda^2} = \frac{1}{\lambda^2}$$

Standard deviation = $1/\lambda = E(X)$. The Exponential's std dev equals its mean — a distinctive feature.

Normal($\mu, \sigma^2$)
$\text{Var}(X) = \sigma^2$

Substitute $z = (x-\mu)/\sigma$, so $x = \mu + \sigma z$:

$$E(X^2) = \int_{-\infty}^\infty (\mu+\sigma z)^2 \frac{1}{\sqrt{2\pi}}e^{-z^2/2}dz = \mu^2\underbrace{\int\frac{1}{\sqrt{2\pi}}e^{-z^2/2}dz}_{=1} + 2\mu\sigma\underbrace{\int z\frac{1}{\sqrt{2\pi}}e^{-z^2/2}dz}_{=0} + \sigma^2\underbrace{\int z^2\frac{1}{\sqrt{2\pi}}e^{-z^2/2}dz}_{=1}$$ $$= \mu^2 + \sigma^2 \implies \text{Var}(X) = \mu^2 + \sigma^2 - \mu^2 = \sigma^2$$

The two parameters of $N(\mu,\sigma^2)$ are exactly the mean and variance. $\mu$ controls location; $\sigma^2$ controls spread.

Properties of Variance

Property 1: For any constant $C \neq E(X)$: $$\text{Var}(X) < E[(X-C)^2]$$ The mean is the uniquely optimal "representative value" — it minimises the expected squared deviation.
Write $X - C = (X - \mu) + (\mu - C)$. Squaring and taking expectation: $$E[(X-C)^2] = E[(X-\mu)^2] + (\mu-C)^2 + 2(\mu-C)\underbrace{E[X-\mu]}_{=0} = \text{Var}(X) + (\mu-C)^2$$ Since $C \neq \mu$, $(\mu-C)^2 > 0$, so $E[(X-C)^2] > \text{Var}(X)$. $\square$
Property 2: $\text{Var}(aX + b) = a^2\,\text{Var}(X)$

Adding a constant ($b$) shifts the distribution but does not change its spread. Scaling by $a$ stretches/shrinks the spread by $a^2$.
$E(aX+b) = aE(X)+b$. Then: $$\text{Var}(aX+b) = E[(aX+b - (aE(X)+b))^2] = E[a^2(X-E(X))^2] = a^2\,\text{Var}(X) \qquad \square$$
🌡️ Temperature Example — Centigrade vs Fahrenheit

If $C$ is a random temperature (in °C) and $F = \frac{9}{5}C + 32$ is the same temperature in Fahrenheit:

$$\text{Var}(F) = \left(\frac{9}{5}\right)^2\text{Var}(C) = \frac{81}{25}\,\text{Var}(C)$$

The constant shift (+32) doesn't change variance at all. The scaling ($\frac{9}{5}$) multiplies variance by $\left(\frac{9}{5}\right)^2$. Standard deviation scales by $\frac{9}{5}$.

In options pricing, variance is the square of volatility. When structurers move between different currency-denominated returns — say converting Nifty returns from INR to USD — the scaling property means you can't just multiply the volatility by the exchange rate factor. You multiply the variance by the square of that factor. This is a common error in cross-currency volatility translation.

$X \sim \text{Poi}(5)$. What is $\text{Var}(X)$?

A
$25$
B
$5$ — for Poisson, mean = variance = $\lambda$
C
$\sqrt{5}$
D
$10$
Answer: B — $5$

For Poisson$(\lambda)$: $E(X) = \lambda$ and $\text{Var}(X) = \lambda$. So both equal 5. This equality of mean and variance is the defining characteristic of the Poisson distribution. In practice, if you observe count data where the variance is much larger than the mean, the Poisson model is inappropriate — you'd need a negative binomial instead.

$X \sim \text{Bin}(20, 0.4)$. What is the standard deviation of $X$?

A
$4.8$ — since $\text{Var}(X) = npq = 4.8$
B
$\sqrt{4.8} \approx 2.19$ — standard deviation is the square root of variance
C
$8$ — since $E(X) = np = 8$
D
$\sqrt{8} \approx 2.83$
Answer: B — $\sqrt{4.8} \approx 2.19$

$\text{Var}(X) = npq = 20 \times 0.4 \times 0.6 = 4.8$. Standard deviation $= +\sqrt{4.8} \approx 2.19$. Don't confuse variance (4.8) with standard deviation ($\sqrt{4.8}$). Option A gives the variance, not the standard deviation.
probability / ch12b / q03★★ Medium

$X$ has mean 10 and variance 4. $Y = 3X - 7$. What is $\text{Var}(Y)$?

A
$4$ — shifting and scaling don't change variance
B
$12$ — $3 \times \text{Var}(X)$
C
$36$ — $3^2 \times \text{Var}(X)$
D
$29$ — $3^2 \times 4 - 7$
Answer: C — $36$

Property 2: $\text{Var}(aX+b) = a^2\text{Var}(X)$. Here $a=3$, $b=-7$: $\text{Var}(3X-7) = 9 \times 4 = 36$. The constant $-7$ has zero effect on variance. Only the scale factor $a=3$ matters, and it enters as $a^2=9$.
probability / ch12b / q04★★★ Hard

For the Exponential distribution, the standard deviation equals the mean. Which value of $\lambda$ gives a standard deviation of 2?

A
$\lambda = 2$ — since std dev $= \lambda$
B
$\lambda = 4$ — since $\text{Var} = 1/\lambda^2$ and $1/4 = 1/\lambda^2$
C
$\lambda = 0.5$ — since $\sigma = 1/\lambda = 2$
D
$\lambda = \sqrt{2}$
Answer: C — $\lambda = 0.5$

For Exp$(\lambda)$: $\text{Var}(X) = 1/\lambda^2$, so $\sigma = 1/\lambda$. Setting $\sigma = 2$: $1/\lambda = 2 \Rightarrow \lambda = 0.5$. The mean is also $1/\lambda = 2$ — confirming that std dev = mean for the Exponential. Option A incorrectly treats std dev as $\lambda$ rather than $1/\lambda$.
probability / ch12b / q05★★★ Hard

Daily Nifty log-returns $X \sim N(0.0005, 0.0004)$. Annual returns are roughly $252X$ (252 trading days). What is the annual variance?

A
$0.0004 \times 252 = 0.1008$
B
$0.0004 \times 252^2 = 25.4016$
C
$0.0004$ — variance doesn't change under scaling
D
$\sqrt{0.0004} \times 252 = 5.04$
Answer: B — $0.0004 \times 252^2$

$\text{Var}(252X) = 252^2 \times \text{Var}(X) = 63504 \times 0.0004 = 25.4016$. Variance scales by $a^2$, not $a$. Option A multiplies variance by 252 (linear scaling — wrong). Option D uses $\sqrt{\text{Var}} \times 252$, which would give annual standard deviation if you were scaling std dev linearly — but the correct annual std dev is $\sqrt{252^2 \times 0.0004} = 252\sqrt{0.0004} = 252 \times 0.02 = 5.04$. So Option D accidentally gives the correct standard deviation — but the question asks for variance.
