A Nifty call option has spot S₀ = ₹50,000, strike K = ₹52,000, maturity T = 0.25 years, risk-free rate r = 6.5%, and volatility σ = 18%. Which is closest to the BSM call price?

C = S₀·N(d₁) − K·e⁻ʳᵀ·N(d₂)
d₁ = [ln(S₀/K) + (r + σ²/2)·T] / σ√T
d₂ = d₁ − σ√T
A
₹520
B
₹890
C
₹1,240
D
₹1,880
Step-by-step solution:

Step 1 — Compute d₁:
ln(50000/52000) ≈ −0.0392
r + σ²/2 = 0.065 + 0.0162 = 0.0812
d₁ = (−0.0392 + 0.0812×0.25) / (0.18×0.5) = −0.210

Step 2 — Compute d₂:
d₂ = −0.210 − 0.09 = −0.299

Step 3 — Normal CDF values:
N(−0.210) ≈ 0.417  |  N(−0.299) ≈ 0.382

Step 4 — Call price:
C = 50,000×0.417 − 52,000×e⁻⁰·⁰¹⁶²⁵×0.382
= 20,850 − 19,610 = ₹1,240 ✓
💡Use (r + σ²/2) in d₁ — the Itô correction is critical
📊d₂ = d₁ − σ√T is always a quick sanity check
🔁Put via parity: P = C − S₀ + K·e⁻ʳᵀ
