Expectation

The single most useful number you can extract from a distribution. Expectation is a weighted average — and from it flows almost every practical quantity in probability and finance: mean returns, option prices, risk measures, and the foundations of the Central Limit Theorem.

Warm-up: Normal is Closed Under Addition

Before we define expectation, a beautiful result that shows why the Normal distribution is so special.

If $X, Y \sim N(0,1)$ are independent, then $X + Y \sim N(0,2)$.

More generally: if $X_i \sim N(\mu_i, \sigma_i^2)$ are independent, then $\displaystyle\sum_{i=1}^n X_i \sim N\!\left(\sum \mu_i,\, \sum \sigma_i^2\right)$.
Use convolution with limits $-\infty$ to $+\infty$ (Normal lives on all of $\mathbb{R}$): $$f_Z(z) = \int_{-\infty}^\infty \frac{1}{\sqrt{2\pi}}e^{-x^2/2} \cdot \frac{1}{\sqrt{2\pi}}e^{-(z-x)^2/2}\,dx = \frac{1}{2\pi}\int_{-\infty}^\infty e^{-\frac{1}{2}(2x^2 - 2zx + z^2)}\,dx$$ Complete the square: $2x^2 - 2zx + z^2 = \left(\sqrt{2}x - \frac{z}{\sqrt{2}}\right)^2 + \frac{z^2}{2}$. So: $$= \frac{1}{2\pi}e^{-z^2/4}\int_{-\infty}^\infty e^{-\frac{1}{2}\left(\sqrt{2}x - \frac{z}{\sqrt{2}}\right)^2}dx$$ Substitute $y = \sqrt{2}x - \frac{z}{\sqrt{2}}$, so $dx = \frac{dy}{\sqrt{2}}$: $$= \frac{1}{2\pi\sqrt{2}}e^{-z^2/4}\int_{-\infty}^\infty e^{-y^2/2}dy = \frac{1}{2\pi\sqrt{2}}e^{-z^2/4}\cdot\sqrt{2\pi} = \frac{1}{\sqrt{2\pi}\cdot\sqrt{2}}\,e^{-\frac{z^2}{2(\sqrt{2})^2}}$$ This is the $N(0,2)$ PDF. $\square$

Definition of Expectation

Expectation is a weighted average of the values a random variable can take, weighted by their probabilities.

Discrete: If $X$ takes values $x_1, x_2, \ldots$ with probabilities $p_1, p_2, \ldots$: $$E(X) = \sum_i p_i\, x_i$$ Continuous: If $X$ has PDF $f(x)$: $$E(X) = \int_{-\infty}^\infty x\, f(x)\, dx \quad \text{(if this integral converges)}$$
$E(X)$ need not be a value $X$ can take. For Bernoulli($p$), $E(X) = p$ — but $X$ only takes values 0 and 1, never $p$ itself (unless $p = 0$ or 1). We use expectation as a measure of central tendency because it is the most mathematically tractable, not because it represents a "typical" outcome in a naive sense.

Expectations of Named Distributions

Bernoulli($p$)
$E(X) = p$

$E(X) = 1\cdot p + 0\cdot(1-p) = p$. Direct from definition.

Binomial($n,p$)
$E(X) = np$

$E(X) = \sum_{k=0}^n k\binom{n}{k}p^k q^{n-k}$. The $k=0$ term vanishes. For $k \geq 1$: $k\binom{n}{k} = n\binom{n-1}{k-1}$. Substituting $j=k-1$:

$$E(X) = np\sum_{j=0}^{n-1}\binom{n-1}{j}p^j q^{n-1-j} = np(p+q)^{n-1} = np$$
Poisson($\lambda$)
$E(X) = \lambda$

$E(X) = \sum_{k=0}^\infty k\frac{e^{-\lambda}\lambda^k}{k!} = \lambda e^{-\lambda}\sum_{k=1}^\infty \frac{\lambda^{k-1}}{(k-1)!} = \lambda e^{-\lambda}\cdot e^\lambda = \lambda$

Geometric($p$)
$E(X) = \dfrac{1}{p}$

Let $S = E(X) = \sum_{k=1}^\infty k\,pq^{k-1} = p + 2pq + 3pq^2 + \cdots$. Then $qS = pq + 2pq^2 + \cdots$. Subtracting:

$$S(1-q) = p + pq + pq^2 + \cdots = \frac{p}{1-q} = 1 \implies S = \frac{1}{p}$$
Uniform$[a,b]$
$E(X) = \dfrac{a+b}{2}$

$E(X) = \int_a^b x\cdot\frac{1}{b-a}dx = \frac{1}{b-a}\cdot\frac{b^2-a^2}{2} = \frac{a+b}{2}$. The midpoint of the interval.

Exp($\lambda$)
$E(X) = \dfrac{1}{\lambda}$

Substitute $z = \lambda x$: $E(X) = \int_0^\infty x\lambda e^{-\lambda x}dx = \frac{1}{\lambda}\int_0^\infty z e^{-z}dz = \frac{1}{\lambda}\Gamma(2) = \frac{1}{\lambda}$

Gamma($\lambda,\alpha$)
$E(X) = \dfrac{\alpha}{\lambda}$

$E(X) = \frac{\lambda^\alpha}{\Gamma(\alpha)}\int_0^\infty x^\alpha e^{-\lambda x}dx = \frac{\lambda^\alpha}{\Gamma(\alpha)}\cdot\frac{\Gamma(\alpha+1)}{\lambda^{\alpha+1}} = \frac{\alpha\Gamma(\alpha)}{\lambda\Gamma(\alpha)} = \frac{\alpha}{\lambda}$

Check: Exp$(\lambda)$ = Gamma$(\lambda,1)$ gives $E = 1/\lambda$. ✓

Beta₁($m,n$)
$E(X) = \dfrac{m}{m+n}$

$E(X) = \frac{1}{B(m,n)}\int_0^1 x^m(1-x)^{n-1}dx = \frac{B(m+1,n)}{B(m,n)} = \frac{\Gamma(m+1)\Gamma(n)}{\Gamma(m+n+1)}\cdot\frac{\Gamma(m+n)}{\Gamma(m)\Gamma(n)} = \frac{m}{m+n}$

Check: $\beta_1(1,1) = U(0,1)$ gives $E = 1/2 = 1/(1+1)$. ✓

Beta₂($m,n$)
$E(X) = \dfrac{m}{n-1}$   (exists only if $n > 1$)

$E(X) = \frac{B(m+1,n-1)}{B(m,n)} = \frac{m}{n-1}$. If $n \leq 1$, the integral diverges — expectation does not exist. Beta₂ with small $n$ is a distribution with heavy enough tails to kill the mean.

Normal($\mu,\sigma^2$)
$E(X) = \mu$

For $N(0,1)$: $E(X) = \frac{1}{\sqrt{2\pi}}\int_{-\infty}^\infty x e^{-x^2/2}dx = 0$ because $xe^{-x^2/2}$ is an odd function and the integral of any odd function over $(-\infty,\infty)$ is zero.

For $N(\mu,\sigma^2)$: substitute $z = (x-\mu)/\sigma$, the mean shifts to $\mu$. (Exercise.)

When Expectation Doesn't Exist

⚠️ The St. Petersburg Paradox

Let $X = 2^k$ with probability $\frac{1}{2^k}$ for $k = 1, 2, 3, \ldots$

First check it's a valid distribution: $\sum_{k=1}^\infty \frac{1}{2^k} = 1$. ✓

Now compute $E(X)$:

$$E(X) = \sum_{k=1}^\infty 2^k \cdot \frac{1}{2^k} = \sum_{k=1}^\infty 1 = \infty$$

The expectation diverges. A fair game based on this distribution would require an infinite entry fee to be fair — yet most people would pay only a few hundred rupees. This is the St. Petersburg Paradox, and it was a major motivator for developing utility theory.

Properties of Expectation

Linearity:
(1) $E(X + c) = E(X) + c$ for any constant $c$
(2) $E(cX) = c\,E(X)$ for any constant $c$
(3) $E(aX + b) = aE(X) + b$
(4) $E(X + Y) = E(X) + E(Y)$ — even if $X$ and $Y$ are dependent
Proof of (1), discrete: If $X$ takes values $x_i$ with probabilities $p_i$, then $Y = X+c$ takes values $x_i + c$ with the same probabilities: $$E(Y) = \sum_i (x_i + c)p_i = \sum_i x_i p_i + c\sum_i p_i = E(X) + c \qquad \square$$ Continuous: $E(X+c) = \int (x+c)f(x)dx = \int xf(x)dx + c\int f(x)dx = E(X) + c$. $\square$
🎲 Illustration — $X \sim \text{Bin}(3, \frac{1}{2})$, $Y = 2X$

$E(X) = np = 3 \times \frac{1}{2} = \frac{3}{2}$.

$Y$ takes values $\{0, 2, 4, 6\}$ with probabilities $\frac{1}{8}, \frac{3}{8}, \frac{3}{8}, \frac{1}{8}$.

$$E(Y) = 0\cdot\frac{1}{8} + 2\cdot\frac{3}{8} + 4\cdot\frac{3}{8} + 6\cdot\frac{1}{8} = \frac{0+6+12+6}{8} = \frac{24}{8} = 3$$

And indeed $3 = 2 \times \frac{3}{2} = 2\,E(X)$. Linearity confirmed directly. ✓

Linearity of expectation is the engine of options pricing. The expected payoff of a portfolio is the sum of expected payoffs of individual legs — regardless of correlation. This is why you can price a complex structured product by decomposing it into vanilla components. The Black-Scholes formula is fundamentally an expected value computation: $C = e^{-rT}E^\mathbb{Q}[\max(S_T - K, 0)]$.

A
$4$

$14$

probability / ch12 / q02★★ Medium

$X \sim \text{Geo}(0.2)$. On average, how many coin tosses are needed to get the first head?

A
$2$ — since $p = 0.2$ means 20% chance each toss
B
$5$ — since $E(X) = 1/p = 1/0.2$
C
$0.2$ — the probability itself
D
$0.8$ — the probability of failure
Answer: B — $5$ tosses

$E(X) = 1/p = 1/0.2 = 5$. If each toss has a 20% chance of success, you expect to wait 5 trials on average. This is the reciprocal relationship between the success probability and the expected waiting time — a direct consequence of the $S - qS$ proof.
probability / ch12 / q03★★ Medium

$X \sim \Gamma(\lambda, \alpha)$. If $\lambda = 2$ and $E(X) = 3$, what is $\alpha$?

A
$\alpha = 3/2$, since $E(X) = \lambda/\alpha$
B
$\alpha = 6$, since $E(X) = \alpha/\lambda$
C
$\alpha = 2/3$
D
$\alpha = 1.5$
Answer: B — $\alpha = 6$

$E(X) = \alpha/\lambda \Rightarrow 3 = \alpha/2 \Rightarrow \alpha = 6$. The shape parameter $\alpha$ controls how many "events" have accumulated — with rate $\lambda=2$, needing mean time 3 means $\alpha=6$ events. Option A has the formula inverted.

$X \sim N(\mu_1, \sigma_1^2)$ and $Y \sim N(\mu_2, \sigma_2^2)$ are independent. What is the distribution of $X + Y$?

A
$N(\mu_1\mu_2,\, \sigma_1^2\sigma_2^2)$ — multiply both parameters
B
$N\!\left(\frac{\mu_1+\mu_2}{2},\, \frac{\sigma_1^2+\sigma_2^2}{2}\right)$ — average the parameters
C
$N\!\left(\mu_1+\mu_2,\, \sigma_1^2+\sigma_2^2\right)$ — means and variances add
D
Not Normal — sums of Normals require identical distributions
Answer: C

For independent Normals: means add and variances add. $X+Y \sim N(\mu_1+\mu_2, \sigma_1^2+\sigma_2^2)$. The distributions don't need to be identical — only independent. This is the general result stated in the lecture. It's why a portfolio of independent Normal returns is also Normal, with mean = sum of means and variance = sum of variances.
probability / ch12 / q05★★★ Hard

A Nifty option has payoff $\max(S_T - K, 0)$ at expiry. The expected payoff under the risk-neutral measure is $E^\mathbb{Q}[\max(S_T-K,0)]$. If $S_T \sim N(100, 25)$ under $\mathbb{Q}$ and $K = 100$, which statement is correct?

A
$E[\max(S_T-100,0)] = E[S_T] - 100 = 0$ — by linearity
B
$E[\max(S_T-100,0)] = \max(E[S_T]-100,0) = 0$ — move $E$ inside $\max$
C
$E[\max(S_T-100,0)] > 0$ — $\max$ is non-linear so linearity does not apply; the option has positive value
D
Expectation cannot be computed since $\max$ is not differentiable
Answer: C

Linearity applies to linear functions: $E(aX+b) = aE(X)+b$. The $\max$ function is non-linear — you cannot move $E$ inside it. $E[\max(S_T-K,0)] \neq \max(E[S_T]-K,0)$. The option has positive expected payoff even when $S_T$ is centred at $K$, because the upside ($S_T > K$) is unbounded while the downside is floored at zero. For $N(100,25)$ at $K=100$: the payoff is $S_T - 100$ when $S_T > 100$ (half the probability) and 0 otherwise — a strictly positive expectation. This is exactly what Black-Scholes computes.
