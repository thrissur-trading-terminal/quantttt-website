Moment Generating Functions

A distribution is characterised by infinitely many moments. The moment generating function (MGF) packages all of them into a single function — and differentiating it at zero pulls out any moment you want. It also provides the cleanest proofs that sums of independent distributions stay within the same family.

Moments of a Distribution

The $r$-th raw moment of $X$: $\mu'_r = E(X^r)$

The $r$-th central moment: $\mu_r = E[(X - E(X))^r]$

Key instances: $\mu'_0 = 1$, $\mu'_1 = E(X)$ (mean), $\mu_1 = 0$, $\mu_2 = \text{Var}(X)$.

$\mu_3$ measures skewness (asymmetry); $\mu_4$ measures kurtosis (peakedness). Two distributions can have the same mean and variance yet differ in shape — higher moments capture this.
Raw and central moments are interconvertible. Since $\mu'_1 = \mu$ (the mean): $$\mu_2 = \mu'_2 - (\mu'_1)^2 \qquad \mu_3 = \mu'_3 - 3\mu'_2\mu'_1 + 2(\mu'_1)^3$$ So knowing all raw moments gives all central moments, and vice versa. Two distributions are identical if and only if all their moments match ($\mu'_r$ for all $r = 1,2,3,\ldots$). Hence the power of a function that generates all moments at once.

Definition of the MGF

The Moment Generating Function (MGF) of $X$ is: $$M_X(t) = E(e^{tX}), \quad \text{for } t \in (-h, h) \text{ for some } h > 0$$ Expanding $e^{tX}$ as a power series and using linearity of expectation: $$M_X(t) = E\!\left[1 + tX + \frac{t^2X^2}{2!} + \cdots\right] = 1 + t\mu'_1 + \frac{t^2}{2!}\mu'_2 + \cdots + \frac{t^k}{k!}\mu'_k + \cdots$$ The $k$-th raw moment $\mu'_k$ is the coefficient of $t^k/k!$ in the expansion.
Extracting moments by differentiation: $$\mu'_k = M_X^{(k)}(0) = \left.\frac{d^k}{dt^k}M_X(t)\right|_{t=0}$$ Differentiating $k$ times and setting $t=0$ kills all but the $\mu'_k$ term.

MGFs of Named Distributions

Bernoulli($p$)
$M(t) = q + pe^t$

$M(t) = e^{t\cdot 0}\cdot q + e^{t\cdot 1}\cdot p = q + pe^t$. Expanding: $1 + pt + \frac{p}{2!}t^2 + \cdots$, so $\mu'_1 = p$, $\mu'_2 = p$.

Binomial($n,p$)
$M(t) = (q + pe^t)^n$

$M(t) = \sum_{x=0}^n \binom{n}{x}(pe^t)^x q^{n-x} = (q+pe^t)^n$ by the binomial theorem.

Verification: $M'(0) = n(q+p)^{n-1}\cdot p = np$. ✓   $M''(0) = n(n-1)p^2 + np = E(X^2)$. ✓

Poisson($\lambda$)
$M(t) = e^{\lambda(e^t - 1)}$

$M(t) = e^{-\lambda}\sum_{x=0}^\infty\frac{(\lambda e^t)^x}{x!} = e^{-\lambda}\cdot e^{\lambda e^t} = e^{\lambda(e^t-1)}$.

Verification: $M'(t) = e^{\lambda(e^t-1)}\cdot\lambda e^t$. At $t=0$: $M'(0) = \lambda$. ✓

Geometric($p$)
$M(t) = \dfrac{pe^t}{1-qe^t}$   (for $t < -\log q$)

$M(t) = \sum_{x=1}^\infty e^{tx}pq^{x-1} = pe^t\sum_{x=1}^\infty(qe^t)^{x-1} = \frac{pe^t}{1-qe^t}$, valid when $qe^t < 1$.

Exp($\lambda$)
$M(t) = \dfrac{\lambda}{\lambda-t}$   (for $t < \lambda$)

$M(t) = \lambda\int_0^\infty e^{tx}e^{-\lambda x}dx = \lambda\int_0^\infty e^{-(\lambda-t)x}dx = \frac{\lambda}{\lambda-t}$, converges iff $t < \lambda$.

Series: $\frac{\lambda}{\lambda-t} = \sum_{k=0}^\infty \frac{t^k}{\lambda^k}$, so $\mu'_k = \frac{k!}{\lambda^k}$.   Check: $\mu'_1 = 1/\lambda$ ✓, $\mu'_2 = 2/\lambda^2$ ✓.

$N(0,1)$
$M(t) = e^{t^2/2}$

Complete the square: $x^2 - 2tx = (x-t)^2 - t^2$. So $M(t) = \frac{e^{t^2/2}}{\sqrt{2\pi}}\int_{-\infty}^\infty e^{-(x-t)^2/2}dx = e^{t^2/2}$.

Expanding $e^{t^2/2}$: all odd moments are 0. $\mu'_2 = 1$, $\mu'_4 = 3$, $\mu'_6 = 15$.   Generally $\mu'_{2k} = (2k-1)!! = 1\cdot3\cdot5\cdots(2k-1)$.

$N(\mu,\sigma^2)$
$M(t) = e^{\mu t + \frac{1}{2}\sigma^2 t^2}$

Write $X = \mu + \sigma Z$ where $Z \sim N(0,1)$. Then $M_X(t) = E(e^{t(\mu+\sigma Z)}) = e^{\mu t}M_Z(\sigma t) = e^{\mu t}\cdot e^{\sigma^2 t^2/2}$.

Expanding confirms $M'(0) = \mu$ and $M''(0) - (M'(0))^2 = \sigma^2$. The MGF encodes both parameters directly.

Properties and Applications

Property 1 — Scaling: $M_{cX}(t) = M_X(ct)$

Property 2 — Independence: If $X \perp Y$: $$M_{X+Y}(t) = E(e^{t(X+Y)}) = E(e^{tX})E(e^{tY}) = M_X(t)\cdot M_Y(t)$$ The MGF of a sum of independent RVs is the product of their MGFs.

Uniqueness Theorem: If two RVs have the same MGF in a neighbourhood of $t=0$, they have the same distribution.
∑ Reproductive Properties via MGF — The Cleanest Proofs

Binomial: $X_i \sim \text{Ber}(p)$ independent. Each has MGF $q+pe^t$. Sum of $n$ independent Bernoullis:

$$M_{X_1+\cdots+X_n}(t) = (q+pe^t)^n$$

By uniqueness, $X_1+\cdots+X_n \sim \text{Bin}(n,p)$. ✓   This is the cleanest proof that Binomial = sum of Bernoullis.

Poisson: $X \sim \text{Poi}(\lambda_1)$, $Y \sim \text{Poi}(\lambda_2)$ independent:

$$M_{X+Y}(t) = e^{\lambda_1(e^t-1)}\cdot e^{\lambda_2(e^t-1)} = e^{(\lambda_1+\lambda_2)(e^t-1)}$$

By uniqueness, $X+Y \sim \text{Poi}(\lambda_1+\lambda_2)$. ✓

Normal: $X_i \sim N(\mu_i,\sigma_i^2)$ independent:

$$M_{\sum X_i}(t) = \prod e^{\mu_i t + \frac{1}{2}\sigma_i^2 t^2} = e^{(\sum\mu_i)t + \frac{1}{2}(\sum\sigma_i^2)t^2}$$

By uniqueness, $\sum X_i \sim N(\sum\mu_i, \sum\sigma_i^2)$. ✓   This is why the Normal is closed under addition — proved definitively in one line.

In options pricing, the log-return distribution's MGF is directly related to the Laplace transform of the payoff. The MGF of Nifty log-returns (approximately Normal) is $e^{\mu t + \frac{1}{2}\sigma^2 t^2}$ — and this is exactly the function you exponentiate to get the risk-neutral price of a binary option. The Black-Scholes formula is a direct consequence of the Normal MGF evaluated at complex values of $t$.

$M_X(t) = e^{3t + 8t^2}$. What distribution does $X$ follow, and what are its mean and variance?

A
Poisson(3); mean = 3, variance = 3
B
$N(3, 16)$; mean = 3, variance = 16
C
$N(3, 8)$; mean = 3, variance = 8
D
Exp(3); mean = 1/3
Answer: B — $N(3,16)$

The Normal MGF is $e^{\mu t + \frac{1}{2}\sigma^2 t^2}$. Matching: $\mu = 3$ and $\frac{1}{2}\sigma^2 = 8 \Rightarrow \sigma^2 = 16$. So $X \sim N(3,16)$, mean 3, variance 16. Option C incorrectly reads $\sigma^2 = 8$ directly from the coefficient without the factor of $\frac{1}{2}$.

For $X \sim \text{Exp}(\lambda)$, the $k$-th raw moment is $\mu'_k = k!/\lambda^k$. What is $E(X^3)$ for $X \sim \text{Exp}(2)$?

A
$3/8$
B
$3/4$
C
$6$
D
$1/8$
Answer: B — $3/4$

$\mu'_3 = 3!/\lambda^3 = 6/8 = 3/4$. The MGF of Exp(2) is $\frac{2}{2-t}$, and expanding as a power series gives coefficients $k!/2^k$ for $t^k/k!$. Option C gives $3! = 6$ without dividing by $\lambda^3$.
probability / ch14 / q03★★ Medium

$X \sim \text{Poi}(3)$ and $Y \sim \text{Poi}(5)$ are independent. What is $P(X+Y = 0)$?

A
$e^{-3} \cdot e^{-5}$ — multiply the individual zero-probabilities
B
$e^{-8}$ — since $X+Y \sim \text{Poi}(8)$
C
$e^{-3} + e^{-5}$
D
Cannot determine without the joint distribution
Answer: B — $e^{-8}$

By the MGF product property: $M_{X+Y}(t) = e^{3(e^t-1)}\cdot e^{5(e^t-1)} = e^{8(e^t-1)}$. By uniqueness, $X+Y \sim \text{Poi}(8)$. So $P(X+Y=0) = e^{-8}$. Note: Option A gives the same numerical answer ($e^{-3}\cdot e^{-5} = e^{-8}$), but for the wrong reason — it assumes independence and multiplies probabilities directly, which happens to work for the zero event but not in general.
probability / ch14 / q04★★★ Hard

$X \sim N(0,1)$. Using the MGF $e^{t^2/2}$, what is $E(X^4)$?

A
$1$ — all even moments of $N(0,1)$ equal 1
B
$2$ — $\mu'_4 = (t^2/2)^2 = t^4/4$ so coefficient is $1/4$... $\times 4! = 6$?
C
$3$ — coefficient of $t^4/4!$ in $e^{t^2/2}$ is 3
D
$0$ — all moments of $N(0,1)$ beyond the second are zero
Answer: C — $3$

Expand $e^{t^2/2} = 1 + \frac{t^2}{2} + \frac{1}{2!}\left(\frac{t^2}{2}\right)^2 + \cdots = 1 + \frac{t^2}{2} + \frac{t^4}{8} + \cdots$. The coefficient of $t^4/4!$ is $\mu'_4$: $\frac{t^4}{8} = \frac{t^4}{4!}\cdot\mu'_4$, so $\mu'_4 = \frac{4!}{8} = \frac{24}{8} = 3$. This is the famous result: the fourth moment of $N(0,1)$ equals 3, meaning the normal distribution has kurtosis $\mu_4/\sigma^4 = 3$. "Excess kurtosis" subtracts this baseline.
probability / ch14 / q05★★★ Hard

$X_1, X_2, \ldots, X_{20} \sim \text{Bin}(5, 0.3)$ are iid. What is the distribution of $S = X_1+\cdots+X_{20}$, and what is $E(S)$?

A
$N(30, 21)$ — by CLT, sum of iid variables is approximately Normal
B
$\text{Bin}(100, 0.3)$; $E(S) = 30$
C
$\text{Bin}(20, 0.3^5)$; $E(S) = 20\times 0.3^5$
D
$\text{Poi}(30)$; Binomial approximates Poisson for large $n$
Answer: B — $\text{Bin}(100, 0.3)$

Each $X_i$ has MGF $(0.7 + 0.3e^t)^5$. The sum of 20 independent copies: $M_S(t) = [(0.7+0.3e^t)^5]^{20} = (0.7+0.3e^t)^{100}$. By uniqueness, $S \sim \text{Bin}(100, 0.3)$. So $E(S) = 100 \times 0.3 = 30$. This is the MGF proof of the Binomial reproductive property in action — exponents simply add. Option A is approximately correct by CLT but not exact. The exact distribution is Bin(100, 0.3).
