Conditional Expectation & Inequalities

Conditioning transforms a number (expectation) into a function. This chapter develops the two towering results of conditional expectation — the Tower Property and the Law of Total Variance — then derives the two most useful probability bounds: Chebyshev's Inequality and Markov's Inequality.

Conditional Expectation

Given joint RVs $(X,Y)$, the conditional expectation of $X$ given $Y=y$ is: $$E(X|Y=y) = \int x\,f_{X|Y}(x|y)\,dx = \int x\,\frac{f(x,y)}{f_Y(y)}\,dx \quad \text{(continuous)}$$ $$E(X|Y=y) = \sum_x x\,\frac{P(X=x,\, Y=y)}{P(Y=y)} \quad \text{(discrete)}$$ Key point: $E(X|Y=y)$ is a function of $y$. As $Y$ varies, this becomes a random variable $E(X|Y)$. When $X \perp Y$: $E(X|Y=y) = E(X)$ for all $y$.

Tower Property (Law of Total Expectation)

$$E[E(X|Y)] = E(X)$$ Averaging the conditional expectation over $Y$ recovers the unconditional expectation.
$$E_Y[E(X|Y=y)] = \int E(X|Y=y)\cdot f_Y(y)\,dy = \int\left[\int x\cdot\frac{f(x,y)}{f_Y(y)}dx\right]f_Y(y)\,dy = \iint x\,f(x,y)\,dx\,dy$$ Switching order of integration (Fubini): $$= \int x\left[\int f(x,y)\,dy\right]dx = \int x\,f_X(x)\,dx = E(X) \qquad \square$$

Law of Total Variance

$$\text{Var}(X) = E[\text{Var}(X|Y)] + \text{Var}[E(X|Y)]$$ "Expected conditional variance" + "Variance of conditional mean" = Total variance.
Since $\text{Var}(X|Y) = E(X^2|Y) - [E(X|Y)]^2$: $$E[\text{Var}(X|Y)] = E[E(X^2|Y)] - E\{[E(X|Y)]^2\} = E(X^2) - E\{[E(X|Y)]^2\}$$ (using the Tower Property on the first term). And $\text{Var}[E(X|Y)] = E\{[E(X|Y)]^2\} - [E(E(X|Y))]^2 = E\{[E(X|Y)]^2\} - [E(X)]^2$. Adding: the $E\{[E(X|Y)]^2\}$ terms cancel: $$E[\text{Var}(X|Y)] + \text{Var}[E(X|Y)] = E(X^2) - [E(X)]^2 = \text{Var}(X) \qquad \square$$
Think of it this way: total uncertainty in $X$ = (average uncertainty within each group of $Y$) + (variation in the group means). In finance: total return variance = average within-regime variance + variance of regime means. The second term is why regimes matter even if each regime is stable individually.

Chebyshev's Inequality

General form: For any RV $X$, constant $C$, and $\varepsilon > 0$: $$P(|X - C| \geq \varepsilon) \leq \frac{E(X-C)^2}{\varepsilon^2}$$ Standard form (take $C = \mu = E(X)$): $$P(|X - \mu| \geq \varepsilon) \leq \frac{\text{Var}(X)}{\varepsilon^2}$$ $k\sigma$ form: $$P(|X - \mu| \geq k\sigma) \leq \frac{1}{k^2}$$
$P(|X-C| \geq \varepsilon) = \int_{\{x:|x-C|\geq\varepsilon\}} f(x)\,dx$. On this region, $\frac{(x-C)^2}{\varepsilon^2} \geq 1$, so: $$\int_{\{|x-C|\geq\varepsilon\}} f(x)\,dx \leq \int_{\{|x-C|\geq\varepsilon\}} \frac{(x-C)^2}{\varepsilon^2}f(x)\,dx \leq \frac{1}{\varepsilon^2}\int_{-\infty}^\infty (x-C)^2 f(x)\,dx = \frac{E(X-C)^2}{\varepsilon^2} \qquad \square$$
Chebyshev is not tight. For $X \sim U[0,1]$: $\mu=\frac{1}{2}$, $\sigma^2=\frac{1}{12}$. Taking $k=2$: $\varepsilon = 2\sigma = 2/\sqrt{12} \approx 0.577$. Chebyshev gives $P(|X-\frac{1}{2}|\geq 0.577) \leq \frac{1}{4}$. But actually $P = 0$ since $X$ lives in $[0,1]$ and $0.577 > 0.5$. The bound is valid but extremely loose.

Chebyshev's power is universality — it works for any distribution with finite variance, without knowing the shape. Tighter bounds require more assumptions.

Markov's Inequality

If $h(X) \geq 0$ for all $x$, and $E[h(X)]$ exists, then for any $\varepsilon > 0$: $$P(h(X) \geq \varepsilon) \leq \frac{E[h(X)]}{\varepsilon}$$ Special case: $P(X \geq k) \leq \frac{E(X^r)}{k^r}$ for any non-negative RV and $r > 0$.
Partition $\Omega = A \cup A^c$ where $A = \{h(X) \geq \varepsilon\}$: $$E[h(X)] = \sum_{x\in A}h(x)p_x + \sum_{x\in A^c}h(x)p_x \geq \sum_{x\in A}h(x)p_x \geq \sum_{x\in A}\varepsilon\cdot p_x = \varepsilon\cdot P(h(X)\geq\varepsilon)$$ (first inequality: $h \geq 0$ on $A^c$; second: $h(x) \geq \varepsilon$ on $A$). Dividing: $P(h(X)\geq\varepsilon) \leq E[h(X)]/\varepsilon$. $\square$
Chebyshev follows from Markov. Set $h(X) = (X-\mu)^2$ and $\varepsilon = k^2\sigma^2$: $$P((X-\mu)^2 \geq k^2\sigma^2) \leq \frac{E(X-\mu)^2}{k^2\sigma^2} = \frac{\sigma^2}{k^2\sigma^2} = \frac{1}{k^2}$$ This is exactly Chebyshev. But Markov cannot be derived from Chebyshev — Markov is strictly more general.
Chebyshev's inequality is the theoretical foundation of risk management. Without knowing the return distribution of a Nifty portfolio — knowing only its mean $\mu$ and variance $\sigma^2$ — Chebyshev guarantees: $P(\text{return deviates from } \mu \text{ by more than } 3\sigma) \leq \frac{1}{9} \approx 11\%$. Every VaR model that says "3-sigma event should occur less than 11% of the time" is using Chebyshev, implicitly.

probability / ch17 / q01★ Easy

$X \sim N(\mu, \sigma^2)$. By Chebyshev, what is the upper bound on $P(|X-\mu| \geq 2\sigma)$?

A
$0.05$ — from Normal tables
B
$\dfrac{1}{4}$ — Chebyshev with $k=2$
C
$\dfrac{1}{2}$ — Chebyshev with $k=2$ uses $1/(2k)$
D
$\sigma^2/4$
Answer: B — $\frac{1}{4}$

Chebyshev: $P(|X-\mu|\geq k\sigma) \leq 1/k^2$. With $k=2$: bound $= 1/4$. The actual probability for $N(\mu,\sigma^2)$ is about 4.5% — much less than 25%. Chebyshev is universal but loose for smooth distributions like the Normal.
probability / ch17 / q02★★ Medium

$E(X) = 10$, $\text{Var}(X) = 4$. Using Chebyshev, find the minimum $k$ such that $P(|X-10|\geq k) \leq 0.04$.

A
$k = 2$
B
$k = 5$
C
$k = 10$
D
$k = 4$
Answer: C — $k = 10$

Chebyshev: $P(|X-\mu|\geq\varepsilon) \leq \text{Var}/\varepsilon^2$. Set $4/\varepsilon^2 \leq 0.04$: $\varepsilon^2 \geq 100$, so $\varepsilon \geq 10$. To guarantee the probability is at most 4%, you need to be 10 units away from the mean — 5 standard deviations. That's how loose the bound is in practice.
probability / ch17 / q03★★ Medium

$X \geq 0$ with $E(X) = 5$. Using Markov's inequality, what is the bound on $P(X \geq 25)$?

A
$1/5$ — Markov: $P(X\geq k) \leq E(X)/k$
B
$1/25$
C
$5/25 = 1/5$ but Markov doesn't apply since $X$ isn't bounded
D
$0$ — $P(X\geq 25)=0$ since $E(X)=5$
Answer: A — $\frac{1}{5}$

Markov: $P(X\geq\varepsilon) \leq E(X)/\varepsilon = 5/25 = 1/5$. Markov applies to any non-negative RV with a finite mean — no bound on $X$ needed. Option D is wrong — having mean 5 doesn't prevent $X$ from occasionally being 25 or more.
probability / ch17 / q04★★★ Hard

Using the Tower Property: $E(X) = E[E(X|Y)]$. If $E(X|Y=1) = 3$, $E(X|Y=2) = 7$, $P(Y=1) = 0.4$, $P(Y=2) = 0.6$, what is $E(X)$?

A
$5$ — simple average of 3 and 7
B
$5.4$ — weighted: $3(0.4) + 7(0.6)$
C
$21$ — product of the two conditional expectations
D
Cannot determine without the full joint distribution
Answer: B — $5.4$

$E(X) = E[E(X|Y)] = E(X|Y=1)\cdot P(Y=1) + E(X|Y=2)\cdot P(Y=2) = 3(0.4)+7(0.6) = 1.2+4.2 = 5.4$. This is the Tower Property in direct action — the weighted average of conditional expectations. Option A is wrong (unweighted). The Tower Property says you need the full distribution of $Y$ as weights.
probability / ch17 / q05★★★ Hard

A Nifty portfolio has mean return $\mu = 12\%$ and $\sigma = 20\%$ per year. Using Chebyshev, what is the guaranteed upper bound on $P(\text{return} \leq -28\%)$?

A
$0.025$ — from Normal distribution, $z$-score analysis
B
$0.10$
C
$\dfrac{1}{16}$ — since $-28\%$ is $2\sigma$ below the mean
D
$\dfrac{1}{4}$ — Chebyshev at $2\sigma$
Answer: C — $\frac{1}{16}$

$-28\% = 12\% - 40\% = \mu - 2\sigma$. So this event requires the return to be at least $2\sigma = 40\%$ below the mean. Chebyshev: $P(|X-\mu|\geq 2\sigma) \leq 1/4$. But we want the one-sided bound $P(X \leq \mu - 2\sigma)$. By symmetry of the Chebyshev bound, $P(\text{one tail}) \leq \frac{1}{2}\cdot\frac{1}{k^2} = \frac{1}{2}\cdot\frac{1}{4} = \frac{1}{8}$. More precisely, using the one-sided Chebyshev: $P(X \leq \mu - k\sigma) \leq \frac{1}{1+k^2} = \frac{1}{5}$ for $k=2$. The simplest safe bound students use is $P(|X-\mu|\geq 2\sigma)/2 = 1/8$. Option D gives the two-sided bound.
