Joint Distributions & Covariance

A single random variable maps each outcome to a number. Two random variables map each outcome to a point in the plane. This chapter builds the full 2D framework — joint distributions, marginals, conditionals, independence — and then introduces covariance and correlation: the tools for measuring how two random variables move together.

Two-Dimensional Random Variables

A 1D random variable is a mapping $X: \Omega \to \mathbb{R}$. A 2D random variable is a pair $(X,Y): \Omega \to \mathbb{R}^2$, assigning two measurements to each outcome.

$\Omega$ = all NSE trading days. $X(\omega)$ = Nifty return on day $\omega$. $Y(\omega)$ = India VIX on day $\omega$. This is a 2D random variable — every trading day produces a point $(X,Y)$ in the plane. We're interested in how these two move together.
$(X,Y)$ is a 2D random variable if $\{\omega : X(\omega) \leq x \text{ and } Y(\omega) \leq y\}$ is an event for all $(x,y) \in \mathbb{R}^2$.

The Joint CDF: $F(x,y) = P(X \leq x,\, Y \leq y)$

Properties: $F(-\infty, y) = F(x,-\infty) = 0$; $F(\infty,\infty) = 1$; and for $x_1 < x_2$, $y_1 < y_2$: $$P(x_1 < X \leq x_2,\; y_1 < Y \leq y_2) = F(x_2,y_2) - F(x_1,y_2) - F(x_2,y_1) + F(x_1,y_1) \geq 0$$ (Inclusion-exclusion on rectangles — subtract two L-shaped regions, add back the doubly-subtracted corner.)

Joint PMF (Discrete) and Joint PDF (Continuous)

Discrete: $p_{ij} = P(X = x_i, Y = y_j)$ with $\sum_i\sum_j p_{ij} = 1$.

Continuous: A function $f(x,y) \geq 0$ such that $\iint f(x,y)\,dx\,dy = 1$, and $P((X,Y)\in A) = \iint_A f(x,y)\,dx\,dy$.
🎲 Joint PMF Example — Sons and Daughters

Let $X$ = number of sons (0,1,2) and $Y$ = number of daughters (0,1,2,3) for a couple.

X\Y 0 1 2 3 $P(X{=}i)$
0 0.010.060.130.05 0.25
1 0.090.040.200.02 0.35
2 0.100.170.100.03 0.40
$P(Y{=}j)$ 0.20 0.27 0.43 0.10 1.00

Row sums give $P(X=i)$ — the marginal distribution of $X$. Column sums give $P(Y=j)$. The entry 0.20 means $P(X=1, Y=2) = 0.20$.

📈 Joint PDF Example — Verification and Marginals

Let $f(x,y) = x + \frac{xy}{3}$ on $[0,1]\times[0,2]$, zero elsewhere. Is this valid?

$$\int_0^1\int_0^2 \left(x + \frac{xy}{3}\right)dy\,dx = \int_0^1\left[2x + \frac{2x}{3}\right]dx = \int_0^1\frac{8x}{3}\,dx = \frac{4}{3}\cdot\frac{1}{2} \cdot 2 = \frac{2}{3}+\frac{1}{3}=1 \checkmark$$

Marginal PDF of $X$: $f_X(x) = \int_0^2\left(x+\frac{xy}{3}\right)dy = 2x + \frac{2x}{3} = \frac{8x}{3}$ for $x \in [0,1]$.

Marginals, Conditionals, Independence

Marginals: $f_X(x) = \int_{-\infty}^\infty f(x,y)\,dy$    $f_Y(y) = \int_{-\infty}^\infty f(x,y)\,dx$

Conditional PDF: $f_{X|Y}(x|y) = \dfrac{f(x,y)}{f_Y(y)}$    (analogous to $P(A|B) = P(A\cap B)/P(B)$)

Independence: $X \perp Y \iff f(x,y) = f_X(x)\,f_Y(y)$ for all $(x,y)$.

Expectation: $E(X+Y) = E(X) + E(Y)$

Linearity of expectation extends to 2D — and critically, independence is not required. $$E(X+Y) = E(X) + E(Y) \quad \text{always}$$ $$E(aX + bY) = aE(X) + bE(Y) \quad \text{always}$$ But: $E(XY) = E(X)E(Y)$ only if $X,Y$ are independent.
$$E(X+Y) = \iint (x+y)f(x,y)\,dx\,dy = \iint xf(x,y)\,dx\,dy + \iint yf(x,y)\,dx\,dy$$ $$= \int x\underbrace{\left[\int f(x,y)\,dy\right]}_{f_X(x)}dx + \int y\underbrace{\left[\int f(x,y)\,dx\right]}_{f_Y(y)}dy = E(X) + E(Y) \qquad \square$$ The inner integral "marginalises out" the other variable.

Covariance

$$\text{Cov}(X,Y) = E[(X-E(X))(Y-E(Y))] = E(XY) - E(X)E(Y)$$ Sign interpretation: positive Cov → $X$ and $Y$ tend to move together; negative → they move oppositely; zero → no linear relationship (but need not be independent).
Example 1 — Coin Tosses: $X$ = Heads, $Y$ = Tails, $n=3$

$X \sim \text{Bin}(3,p)$, $Y = 3-X$ (deterministically). $E(X) = 3p$, $E(Y) = 3q$.

$XY$ takes values 0 (when $X=0$ or $X=3$) and 2 (when $X=1,Y=2$ or $X=2,Y=1$):

$$E(XY) = 0\cdot(p^3+q^3) + 2\cdot(3pq^2 + 3p^2q) = 6pq(p+q) = 6pq$$ $$\text{Cov}(X,Y) = 6pq - 9pq = -3pq$$

Negative — as expected. More heads means fewer tails. The two are negatively related by construction.

Example 2 — $Y = k$ (constant): $\text{Cov} = 0$

If $Y$ is a constant regardless of $X$, they're independent. $E(XY) = kE(X) = E(X)E(Y)$. So $\text{Cov} = 0$.

Example 3 — $Y = X+1$: $\text{Cov} = 3pq$ (positive)

$E(Y) = E(X)+1 = 3p+1$. $E(XY) = E(X(X+1)) = E(X^2)+E(X) = 6pq+12p$. So $\text{Cov} = 6pq+12p - 3p(3p+1) = 3pq > 0$. As $X$ increases, $Y$ increases — positive covariance.

Variance of a Linear Combination

$$\text{Var}(aX + bY) = a^2\text{Var}(X) + b^2\text{Var}(Y) + 2ab\,\text{Cov}(X,Y)$$ If $X_1,\ldots,X_n$ are independent: $$\text{Var}\!\left(\sum_{i=1}^n a_i X_i\right) = \sum_{i=1}^n a_i^2\,\text{Var}(X_i)$$
Let $\mu_X = E(X)$, $\mu_Y = E(Y)$. Then $aX+bY - E(aX+bY) = a(X-\mu_X)+b(Y-\mu_Y)$. Squaring and taking $E$: $$\text{Var}(aX+bY) = a^2E[(X-\mu_X)^2] + b^2E[(Y-\mu_Y)^2] + 2ab\,E[(X-\mu_X)(Y-\mu_Y)]$$ $$= a^2\text{Var}(X) + b^2\text{Var}(Y) + 2ab\,\text{Cov}(X,Y) \qquad \square$$
Variance consistency check: $X \sim \Gamma(\lambda,\alpha)$ and $Y \sim \Gamma(\lambda,\beta)$ independent. Two routes:
Route 1: $\text{Var}(X+Y) = \alpha/\lambda^2 + \beta/\lambda^2 = (\alpha+\beta)/\lambda^2$ (add variances).
Route 2: $X+Y \sim \Gamma(\lambda,\alpha+\beta)$, so $\text{Var} = (\alpha+\beta)/\lambda^2$. Same answer. ✓

Correlation Coefficient

$$\rho_{XY} = \frac{\text{Cov}(X,Y)}{\sigma_X\,\sigma_Y}$$ Dimensionless and unit-free. Covariance is affected by the units of $X$ and $Y$ (kg·cm ≠ kg·m). Correlation is not — it always lives in $[-1,+1]$.
$-1 \leq \rho_{XY} \leq 1$
Let $U = \frac{X-\mu_X}{\sigma_X}$ and $V = \frac{Y-\mu_Y}{\sigma_Y}$. Then $E(U)=E(V)=0$, $\text{Var}(U)=\text{Var}(V)=1$, and $E(UV) = \rho_{XY}$.

Since $E[(U-V)^2] \geq 0$: $$E[U^2] + E[V^2] - 2E[UV] \geq 0 \implies 1 + 1 - 2\rho_{XY} \geq 0 \implies \rho_{XY} \leq 1$$ Since $E[(U+V)^2] \geq 0$: $$1 + 1 + 2\rho_{XY} \geq 0 \implies \rho_{XY} \geq -1 \qquad \square$$
Invariance under change of origin and scale: If $U = (X-a)/h$ and $V = (Y-b)/k$, then: $$\rho_{UV} = \begin{cases}\rho_{XY} & \text{if } h \text{ and } k \text{ have the same sign} \\ -\rho_{XY} & \text{if opposite signs}\end{cases}$$ Shifting origin doesn't affect correlation. Scaling doesn't either — unless one axis is reflected (sign flip).
The Nifty-VIX correlation is typically around $-0.7$ to $-0.8$ — strongly negative. When Nifty falls, VIX rises. This is why long Nifty + long VIX futures is a natural hedging structure: their covariance term reduces portfolio variance. The correlation coefficient is the precise tool for quantifying this relationship independent of whether you measure returns in basis points or percentages.

$X$ and $Y$ are independent with $E(X) = 3$, $E(Y) = 5$. What is $E(2X + 3Y - 1)$?

A
$20$
B
$20$ — wait, let me check: $2(3)+3(5)-1 = 6+15-1 = 20$
C
$21$
D
Depends on the joint distribution
Answer: B — $20$

$E(2X+3Y-1) = 2E(X)+3E(Y)-1 = 6+15-1 = 20$. Linearity of expectation holds regardless of whether $X$ and $Y$ are independent. Option D is wrong — linearity works for any joint distribution. (Options A and B both say 20 — B is correct with the explicit calculation shown.)
probability / ch13 / q02★★ Medium

$X$ and $Y$ are independent, $\text{Var}(X) = 4$, $\text{Var}(Y) = 9$. What is $\text{Var}(X - Y)$?

A
$5$ — subtract the variances
B
$-5$ — $\text{Var}(X) - \text{Var}(Y)$
C
$13$ — $\text{Var}(X) + \text{Var}(Y)$ since independent
D
$36$ — $(\text{Var}(X) + \text{Var}(Y))^2/2$
Answer: C — $13$

$\text{Var}(X-Y) = \text{Var}(X) + (-1)^2\text{Var}(Y) + 2(1)(-1)\text{Cov}(X,Y) = 4 + 9 + 0 = 13$. Variance is always non-negative — you can never get $-5$. Subtracting a random variable adds its variance. This surprises many people: the difference $X-Y$ is more spread out than either $X$ or $Y$ alone.
probability / ch13 / q03★★ Medium

$\text{Cov}(X,Y) = 0$. Which conclusion is correct?

A
$X$ and $Y$ are independent
B
$X$ and $Y$ have no linear relationship, but may still be dependent
C
$E(XY) = 0$
D
$\rho_{XY} = -1$
Answer: B

Zero covariance means no linear relationship — but $X$ and $Y$ could still be dependent in a non-linear way. Classic example: $X \sim U(-1,1)$, $Y = X^2$. Then $\text{Cov}(X,Y) = E(X^3) - E(X)E(X^2) = 0 - 0 = 0$, yet $Y$ is a deterministic function of $X$ — maximally dependent, yet zero covariance. Independence implies zero covariance, but not the other way around.
probability / ch13 / q04★★★ Hard

$\text{Var}(X) = 4$, $\text{Var}(Y) = 9$, $\rho_{XY} = 0.5$. What is $\text{Var}(X + Y)$?

A
$13$ — variances always add
B
$169$ — $(2+3)^2$
C
$19$ — $\text{Var}(X)+\text{Var}(Y)+2\text{Cov}(X,Y) = 4+9+6$
D
$25$
Answer: C — $19$

$\text{Cov}(X,Y) = \rho\,\sigma_X\sigma_Y = 0.5 \times 2 \times 3 = 3$. Then $\text{Var}(X+Y) = 4 + 9 + 2(3) = 19$. When variables are positively correlated, their sum is more spread out than if they were independent ($13$). In portfolio terms: two positively correlated assets give higher portfolio variance than two independent ones.
probability / ch13 / q05★★★ Hard

Nifty returns $X$ and VIX changes $Y$ have $\rho_{XY} = -0.75$. You convert both to USD (multiply each by the same exchange rate $e > 0$). What is the new correlation?

A
$-0.75/e$ — correlation scales inversely with the factor
B
$-0.75 \times e^2$ — variance scales by $e^2$, so correlation does too
C
$-0.75$ — correlation is invariant under scaling by the same positive factor
D
$+0.75$ — multiplying by $e$ flips the sign
Answer: C — $-0.75$

Let $U = eX$, $V = eY$ (same scale factor, same sign). Then $\text{Cov}(U,V) = e^2\text{Cov}(X,Y)$, $\sigma_U = e\sigma_X$, $\sigma_V = e\sigma_Y$. So $\rho_{UV} = \frac{e^2\text{Cov}}{e\sigma_X \cdot e\sigma_Y} = \rho_{XY} = -0.75$. Correlation is dimensionless and unit-invariant — the $e^2$ factors cancel completely. Only if you applied a negative factor to one of them (e.g. $-e$ for one) would the sign flip.
