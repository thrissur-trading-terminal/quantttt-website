Markov Chains

A system with no memory of its past — only its current state matters for its future. Surprisingly, this simple restriction gives rise to a rich theory that underlies everything from Google's PageRank to options pricing models to credit rating migrations.

Stochastic Processes: The Big Picture

A stochastic process $\{X(t),\, t \in T\}$ is a collection of random variables indexed by time $T$. At each time $t$, the process is in some state $X(t)$.

  • Discrete-time: $T = \{0, 1, 2, \ldots\}$ — written $\{X_n : n \ge 0\}$
  • Continuous-time: $T = [0, \infty)$ — written $\{X(t) : t \ge 0\}$

The state space is the set of all values $X(t)$ can take. This chapter focuses on discrete-time processes with a countable state space — the setting of Markov chains.

A credit rating (AAA, AA, A, BBB, ..., D) changes over time. Today's rating depends on the company's current financial health, not on its entire history. This is the Markov assumption — and it makes credit rating migration matrices a direct application of Markov chain theory. Banks use exactly this framework to estimate default probabilities over multi-year horizons.

The Markov Property: Only Now Matters

A discrete-time stochastic process $\{X_n : n \ge 0\}$ with countable state space is a Markov chain if for all states $i, j$ and all times $n$: $$P(X_{n+1} = j \mid X_n = i,\, X_{n-1} = i_{n-1},\, \ldots,\, X_0 = i_0) = P(X_{n+1} = j \mid X_n = i) = P_{ij}$$ The future depends on the present, not the past. The numbers $P_{ij}$ are called transition probabilities.
Imagine you're playing chess. The Markov property says: the best next move depends only on the current board position, not on the sequence of moves that led there. Two positions that look identical are identical for future purposes — history is irrelevant. This is a strong assumption, but it's surprisingly useful in practice.

The transition probabilities must satisfy:

$$P_{ij} \ge 0 \quad \forall i, j \qquad \text{and} \qquad \sum_{j=0}^\infty P_{ij} = 1 \quad \forall i$$

Every row sums to 1 — from any state, you must go somewhere. The matrix of all $P_{ij}$ is the transition probability matrix $P$.

Three Classic Examples

Example 1 — Weather (1-day memory): States: 0 = raining, 1 = not raining. If it rains today, it rains tomorrow with probability $\alpha$. If it doesn't rain today, it doesn't rain tomorrow with probability $\beta$.

$$P = \begin{pmatrix} \alpha & 1-\alpha \\ 1-\beta & \beta \end{pmatrix} \quad \leftarrow \begin{matrix} \text{from state 0} \\ \text{from state 1} \end{matrix}$$

Example 2 — Gambler's Ruin (Random Walk on $\{0, 1, \ldots, N\}$): At each step, move right with probability $p$, left with probability $1-p$. States 0 and $N$ are absorbing — once you reach them, you stay.

$$P_{00} = 1, \quad P_{NN} = 1 \quad \text{(absorbing boundaries)}$$ $$P_{i,i+1} = p, \quad P_{i,i-1} = 1-p \quad \text{for } i = 1, \ldots, N-1$$

Example 3 — Random Walk on $\mathbb{Z}$: Step right with probability $p$, left with probability $1-p$, on all integers. State space is infinite.

A simplified model of a market maker's inventory: they start with inventory $i$, each trade either buys from them (moves to $i-1$) or sells to them (moves to $i+1$). The inventory follows a random walk. The absorbing barriers model inventory limits — when inventory hits 0 or $N$, the market maker stops quoting one side.

Chapman-Kolmogorov Equations

How do we compute the probability of going from state $i$ to state $j$ in $n$ steps? We need $n$-step transition probabilities $P_{ij}^{(n)} = P(X_n = j \mid X_0 = i)$.

Chapman-Kolmogorov Equations: $$P_{ij}^{(m+n)} = \sum_{k=0}^\infty P_{ik}^{(n)} P_{kj}^{(m)}$$ In matrix form: $P^{(n)} = P^n$ (the $n$-th power of the transition matrix).
To go from $i$ to $j$ in $m+n$ steps, pass through some intermediate state $k$ at step $n$: $$P_{ij}^{(m+n)} = \sum_k P(X_{m+n}=j, X_n=k \mid X_0=i)$$ $$= \sum_k P(X_{m+n}=j \mid X_n=k)\, P(X_n=k \mid X_0=i) = \sum_k P_{kj}^{(m)} P_{ik}^{(n)} \quad \square$$ The Markov property lets us split the conditioning at time $n$.

This means: the $(i,j)$ entry of $P^n$ gives the probability of going from state $i$ to state $j$ in exactly $n$ steps. To find the state distribution at time $n$:

$$\alpha^{(n)} = \alpha^{(0)} P^n$$

where $\alpha^{(0)}$ is the row vector of initial probabilities.

🎱 Worked Example — Ball Replacement

A box has 2 balls. At each stage, pick one randomly and replace it: same colour with prob 0.7, different with prob 0.3. Start: both balls red. What's the probability the 5th selection is red?

State $X_n$ = number of red balls. State space $\{0, 1, 2\}$.

$$P = \begin{pmatrix} 0.7 & 0.3 & 0 \\ 0.15 & 0.7 & 0.15 \\ 0 & 0.3 & 0.7 \end{pmatrix}$$

Start: $X_0 = 2$, so $\alpha^{(0)} = [0, 0, 1]$. Compute $\alpha^{(5)} = \alpha^{(0)} P^5$. Then:

$$P(\text{5th is red}) = 0 \cdot P_{20}^{(5)} + \tfrac{1}{2} \cdot P_{21}^{(5)} + 1 \cdot P_{22}^{(5)}$$

Classification of States

Not all states behave the same way. The long-run behaviour of a Markov chain depends critically on which states are "reachable" from which others.

State $j$ is accessible from state $i$ (written $i \to j$) if $P_{ij}^{(n)} > 0$ for some $n \ge 1$.

States $i$ and $j$ communicate (written $i \leftrightarrow j$) if $i \to j$ and $j \to i$.

Communication is an equivalence relation — it partitions the state space into communicating classes.

A Markov chain is irreducible if all states communicate (one single class).

Within each communicating class, every state is either recurrent or transient — recurrence is a class property.

Recurrent: returns infinitely often Transient: eventually leaves forever Absorbing: stays once entered ($P_{ii}=1$)
State $i$ is recurrent if $\displaystyle\sum_{n=0}^\infty P_{ii}^{(n)} = \infty$, and transient if $\displaystyle\sum_{n=0}^\infty P_{ii}^{(n)} < \infty$.

Equivalently: $i$ is recurrent if the process returns to $i$ with probability 1; transient if there's a positive probability of never returning.
Think of it like a city. A recurrent state is a city you'll definitely return to eventually — maybe not soon, but with probability 1. A transient state is a city you pass through: you might never go back. In a finite irreducible chain, all states are recurrent — there's nowhere to escape to permanently.
🚶 Worked Example — Random Walk on ℤ

For the simple random walk on $\mathbb{Z}$ (step right with prob $p$, left with prob $1-p$), is state 0 recurrent?

$P_{00}^{(2n-1)} = 0$ (can't return to 0 in odd steps). For even steps, using combinatorics:

$$P_{00}^{(2n)} = \binom{2n}{n} p^n (1-p)^n$$

By Stirling's approximation: $P_{00}^{(2n)} \sim \dfrac{(4p(1-p))^n}{\sqrt{\pi n}}$

When $p = 1/2$: $4p(1-p) = 1$, so $P_{00}^{(2n)} \sim \dfrac{1}{\sqrt{\pi n}}$, and $\sum_{n=1}^\infty \dfrac{1}{\sqrt{\pi n}} = \infty$.

State 0 is recurrent when $p = 1/2$. When $p \neq 1/2$: $4p(1-p) < 1$, geometric decay, sum is finite — state 0 is transient. A biased random walk drifts to infinity and never returns.

Periodicity

Even in a recurrent chain, limits might not exist if the chain has a periodic structure. The period of state $i$ is $d(i) = \gcd\{n \ge 1 : P_{ii}^{(n)} > 0\}$. A state is aperiodic if $d(i) = 1$.

Consider the chain with states $\{0,1\}$ and $P = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}$. It bounces between 0 and 1 forever. $P^n = I$ for even $n$, $P^n = P$ for odd $n$ — no limit exists. This chain has period 2. The stationary distribution $\beta = [1/2,\, 1/2]$ exists, but $\lim_{n\to\infty} P^n$ does not.

Stationary and Limiting Distributions

A distribution $\beta = [\beta_0, \beta_1, \ldots]$ is a stationary distribution if starting from $\beta$, the chain stays in $\beta$ forever:

$$\beta_j = \sum_{i=0}^\infty \beta_i P_{ij} \quad \forall j \qquad \text{and} \qquad \sum_{i=0}^\infty \beta_i = 1$$ $$\text{Matrix form: } \beta = \beta P$$
Ergodic Theorem (Perron-Frobenius): If a Markov chain is irreducible and aperiodic (ergodic), then:
  • A unique stationary distribution $\beta$ exists
  • $\lim_{n\to\infty} P_{ij}^{(n)} = \beta_j$ for all $i, j$ — regardless of starting state
  • $\beta_j = 1/\mu_j$ where $\mu_j$ is the mean return time to state $j$
🌦 Worked Example — Weather Stationary Distribution

Weather chain with $P = \begin{pmatrix} 2/3 & 1/3 \\ 1/2 & 1/2 \end{pmatrix}$. Find the stationary distribution.

Solve $\beta P = \beta$ with $\beta_0 + \beta_1 = 1$:

$$\beta_0 = \frac{2}{3}\beta_0 + \frac{1}{2}\beta_1 \implies \frac{1}{3}\beta_0 = \frac{1}{2}\beta_1 \implies \beta_0 = \frac{3}{2}\beta_1$$ $$\beta_0 + \beta_1 = 1 \implies \frac{3}{2}\beta_1 + \beta_1 = 1 \implies \beta_1 = \frac{2}{5}, \quad \beta_0 = \frac{3}{5}$$

In the long run, it rains 60% of days. Notice: computing $P^5$ and $P^{10}$ from the class notes gave rows converging to $[0.6, 0.4]$ — exactly this stationary distribution.

Credit rating agencies (CRISIL, ICRA) publish annual rating transition matrices. If you treat credit ratings as a Markov chain, the stationary distribution tells you the long-run fraction of issuers at each rating level. More importantly, $P^{10}$ gives you the 10-year default probability starting from any current rating — directly useful for pricing long-dated bonds and credit derivatives.

The Left-Skip-Free Random Walk

A beautiful application of conditioning: let $X_1, X_2, \ldots$ be iid with values in $\{-1, 0, 1, 2, \ldots\}$ and mean $E(X_i) = -\nu < 0$. Define $S_0 = 0$, $S_n = \sum_{i=1}^n X_i$. This is a left-skip-free random walk — it can drop by at most 1 at each step.

Let $T_{-k} = \min\{n : S_n = -k\}$ be the hitting time of level $-k$. By the iid structure, hitting times have a beautiful additive decomposition:

$$E(T_{-k}) = \frac{k}{\nu} \qquad \text{Var}(T_{-k}) = \frac{k\sigma^2}{\nu^3}$$

where $\sigma^2 = \text{Var}(X_i)$. The mean hitting time scales linearly in $k$, and the variance also scales linearly — the walk reaches level $-k$ in roughly $k/\nu$ steps, with fluctuations of order $\sqrt{k}$.

Remember: row = current state, column = next state. From state 0, you go to state 0 with prob 0.7 and to state 1 with prob 0.3. The transition matrix is read row by row — each row is a probability distribution over next states.

For the weather chain $P = \begin{pmatrix} 2/3 & 1/3 \\ 1/2 & 1/2 \end{pmatrix}$, what is the stationary distribution $[\beta_0, \beta_1]$ (rain, no rain)?

A
$[1/2,\; 1/2]$
B
$[2/3,\; 1/3]$
C
$[3/5,\; 2/5]$
D
$[1/3,\; 2/3]$
Answer: C — $[3/5,\; 2/5]$

Solve $\beta P = \beta$ with $\beta_0 + \beta_1 = 1$. From the first equation: $$\beta_0 = \tfrac{2}{3}\beta_0 + \tfrac{1}{2}\beta_1 \implies \tfrac{1}{3}\beta_0 = \tfrac{1}{2}\beta_1 \implies \beta_0 = \tfrac{3}{2}\beta_1$$ Substituting: $\tfrac{3}{2}\beta_1 + \beta_1 = 1 \implies \beta_1 = 2/5,\; \beta_0 = 3/5$.

stochastic / ch03 / q03 ★★ Medium

A Markov chain on $\{0, 1, 2\}$ has $P_{00} = 1$ (absorbing) and all other states communicate. Which statement is correct?

A
The chain is irreducible
B
All states are recurrent
C
State 0 is recurrent (absorbing), states 1 and 2 are transient
D
State 0 is transient because it can't leave
Answer: C

State 0 is absorbing ($P_{00}=1$) — once entered, the chain stays there forever. It returns to itself with probability 1, so it's recurrent. States 1 and 2 communicate with each other but not with state 0 (0 can't be reached from 1 or 2 because once you leave toward 0, state 0 is absorbing). Since 0 is a "black hole," the chain eventually gets absorbed there — so states 1 and 2 are transient.

The chain is not irreducible — there are three communicating classes: $\{0\}$, and $\{1,2\}$ (if they communicate). Absorbing states are always recurrent — they never leave.
stochastic / ch03 / q04 ★★ Medium

A credit rating model has 3 states: Good (G), Watch (W), Default (D). Default is absorbing. Starting in state G, after many years, what is $\lim_{n\to\infty} P(X_n = G \mid X_0 = G)$?

A
Depends on the exact transition probabilities
B
$0$ — Default is absorbing, so eventually all mass moves to D
C
$1$ — the chain always returns to G
D
The stationary distribution value of G
Answer: B — 0

Since Default (D) is an absorbing state and G and W are transient (they can reach D but D can never reach them), the chain is eventually absorbed into D with probability 1. As $n \to \infty$, all probability mass accumulates at D.

So $\lim_{n\to\infty} P(X_n = G \mid X_0 = G) = 0$ and $\lim_{n\to\infty} P(X_n = D \mid X_0 = G) = 1$.

This is why long-dated corporate bonds always have positive default probabilities — no matter how good the rating today, given enough time, all transient states drain into absorbing ones.
stochastic / ch03 / q05 ★★★ Hard

An ergodic Markov chain has stationary distribution $\beta = [0.2, 0.5, 0.3]$. What is the mean return time to state 1 (the middle state)?

A
$0.5$ steps
B
$2$ steps
C
$5$ steps
D
$1/0.2 = 5$ steps
Answer: B — $2$ steps

For an ergodic Markov chain, the stationary probability of state $j$ equals the reciprocal of the mean return time to $j$: $$\beta_j = \frac{1}{\mu_j} \implies \mu_j = \frac{1}{\beta_j}$$ State 1 has $\beta_1 = 0.5$, so the mean return time is $\mu_1 = 1/0.5 = 2$ steps.

Option D uses $\beta_0 = 0.2$ by mistake. High stationary probability = short mean return time. If you're in a state 50% of the time long-run, on average you return every 2 steps.
