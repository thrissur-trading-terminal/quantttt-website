Random Variables & The CDF

Sample spaces made of symbols — H, T, HHT, THH — are useless for calculation. A random variable translates them into numbers. Once on the real line, the full machinery of calculus, algebra, and analysis becomes available. This chapter builds that bridge.

Why Random Variables?

Toss a coin 100 times. Your sample space $\Omega$ has $2^{100}$ elements — strings like HHTHTTHT… — each one a sequence of 100 symbols. Nobody cares about any specific string. What we actually want to know is: how many heads appeared? Or: how many tosses until the first head? Or: how many tosses to get 10 heads?

These questions all have the same structure — they collapse a complicated symbolic outcome into a single number that captures what matters. That is exactly what a random variable does.

A random variable is not random. And it is not a variable in the algebra sense. It is a function — a deterministic rule that takes an outcome $\omega \in \Omega$ and returns a real number $X(\omega) \in \mathbb{R}$. The "random" comes from the fact that $\omega$ itself is random. The function is fixed; randomness lives in the input.
🏭 Manufacturing — Why This Matters Practically

A machine produces 100 items. Each item is either defective (like Tails) or good (like Heads). We model this as 100 coin tosses. The sample space has $2^{100}$ strings — one per possible sequence of good/defective items.

But the factory owner doesn't care which specific sequence occurred. They care about one number: how many items are non-defective? The random variable $X$ = "number of non-defective items" collapses all $2^{100}$ strings into a single integer from 0 to 100. That integer is what drives every business decision.

The Formal Definition

Given a probability space $(\Omega, \mathcal{S}, P)$, a random variable $X$ is a function $X: \Omega \to \mathbb{R}$ such that for every $x \in \mathbb{R}$: $$\{\omega \in \Omega \mid X(\omega) \leq x\} \in \mathcal{S}$$ That is, the set of all outcomes that $X$ maps to values $\leq x$ must be a valid event — it must belong to the sigma-field $\mathcal{S}$.

Why this condition? Because we want to be able to compute $P(X \leq x)$ for any real $x$. If $\{\omega \mid X(\omega) \leq x\}$ is not in $\mathcal{S}$, then $P$ is not defined on it — and the whole framework collapses. The measurability condition ensures that every natural question about $X$ has a well-defined probability.

There is no probability in the definition of a random variable. The definition only involves $\Omega$, $\mathcal{S}$, and the mapping rule. Probability enters only later when we compute things like $P(X \leq x)$ or $P(a < X \leq b)$. This separation is deliberate and important.
🪙 Three Coin Tosses — X = Number of Heads

$\Omega = \{HHH, HHT, HTH, HTT, THH, THT, TTH, TTT\}$

Define $X(\omega)$ = number of H's in $\omega$:

$X(HHH) = 3$  |  $X(HHT) = X(HTH) = X(THH) = 2$  |  $X(HTT) = X(THT) = X(TTH) = 1$  |  $X(TTT) = 0$

Now the events $(X \leq 2)$, $(1 \leq X \leq 3)$, $(X \geq 2)$ are all valid events in $\mathcal{S}$ — each corresponds to a union of elementary outcomes, and we can compute their probabilities directly.

Algebra of Random Variables

One of the most useful facts about random variables: they're closed under the operations you'd expect.

Sum / Difference
If $X_1, X_2$ are RVs, so are $X_1 + X_2$ and $X_1 - X_2$
Product
If $X_1, X_2$ are RVs, so is $X_1 X_2$
Linear Combination
$c_1 X_1 + c_2 X_2$ is an RV for any constants $c_1, c_2 \in \mathbb{R}$
Reciprocal
$\frac{1}{X}$ is an RV (where $X \neq 0$)
Absolute Value
$|X|$ is an RV
Min / Max
$\min(X_1,\ldots,X_n)$ and $\max(X_1,\ldots,X_n)$ are RVs
In options pricing, almost everything is a random variable built from other random variables. The payoff of a call option is $\max(S_T - K, 0)$ — that's a max of two RVs. The P&L of a delta-hedged portfolio is a sum of RVs. The Heston model's variance process is an RV driven by a Brownian motion (also an RV). The algebra of RVs guarantees all these are well-defined and measurable.

The Cumulative Distribution Function

Every random variable comes with a canonical companion: its CDF. It packages everything about the distribution of $X$ into a single function.

The Cumulative Distribution Function (CDF) of a random variable $X$ is the function $F: \mathbb{R} \to [0,1]$ defined by: $$F(x) = P(X \leq x), \quad x \in \mathbb{R}$$ If $X$ and $Y$ are different RVs, we write $F_X$ and $F_Y$ (or $F$ and $G$) to distinguish their CDFs.

Properties of the CDF

1
Bounded

$0 \leq F(x) \leq 1$ for all $x \in \mathbb{R}$.

Obvious — $F(x) = P(X \leq x)$ is a probability, so it lives in $[0,1]$ by definition.

2
Interval Probabilities

For $a < b$:

$$P(a < X \leq b) = F(b) - F(a)$$
The event $(X \leq b)$ can be written as the disjoint union $(X \leq a) \cup (a < X \leq b)$. By countable additivity: $P(X \leq b) = P(X \leq a) + P(a < X \leq b)$. Rearranging gives the result. $\square$

$P(a \leq X \leq b) = P(X=a) + F(b) - F(a)$

$P(a < X < b) = F(b) - F(a) - P(X=b)$

Key consequence: If $P(X=a) = 0$ and $P(X=b) = 0$ — as in all continuous distributions — then all four interval variants $(a,b]$, $[a,b]$, $(a,b)$, $[a,b)$ have the same probability $F(b)-F(a)$. The endpoints simply don't matter.

3
Monotone Non-Decreasing

If $a < b$, then $F(a) \leq F(b)$.

Immediate from Property 2: $F(b) - F(a) = P(a < X \leq b) \geq 0$. $\square$
4
Boundary Behaviour
$$\lim_{x \to -\infty} F(x) = 0 \qquad\qquad \lim_{x \to +\infty} F(x) = 1$$

The CDF starts at 0 (far left, no outcomes below) and ends at 1 (far right, all outcomes below).

Partition $\mathbb{R}$ into countably many disjoint intervals: $\cdots\cup(-2,-1]\cup(-1,0]\cup(0,1]\cup(1,2]\cup\cdots$ By countable additivity: $$1 = P(-\infty < X < \infty) = \sum_i P(i-1 < X \leq i) = \sum_i [F(i) - F(i-1)]$$ This telescopes. For any finite truncation: $$\sum_{i=-a}^{b} [F(i) - F(i-1)] = F(b) - F(-a)$$ Taking $a,b \to \infty$: $F(+\infty) - F(-\infty) = 1$. Since $0 \leq F \leq 1$ everywhere, the only consistent solution is $F(+\infty) = 1$ and $F(-\infty) = 0$. $\square$

Two Types of CDF — Discrete vs Continuous

The CDF can look very different depending on whether $X$ takes discrete values or varies continuously.

Left: Bernoulli RV — jumps of size $p$ at $x=0$ and $1-p$ at $x=1$. Right: Uniform$[0,1]$ RV — smooth ramp, no discontinuities.

🪙 Example 1 — Bernoulli CDF

Let $X(H) = 1$ with probability $1-p$, and $X(T) = 0$ with probability $p$. Then:

$$F(x) = \begin{cases} 0 & x < 0 \\ p & 0 \leq x < 1 \\ 1 & x \geq 1 \end{cases}$$

The jumps in $F$ occur exactly at the points where $X$ has positive probability mass (0 and 1). The jump size equals the probability at that point. This is the hallmark of a discrete random variable.

📏 Example 2 — Uniform[0,1] CDF

Let $X$ be uniform on $[0,1]$: its density is 1 on $[0,1]$ and 0 elsewhere. Then:

$$F(x) = \begin{cases} 0 & x < 0 \\ x & 0 \leq x \leq 1 \\ 1 & x > 1 \end{cases}$$

$F$ is perfectly continuous — no jumps anywhere. This is the hallmark of a continuous random variable. Any individual point has probability zero: $P(X = 0.5) = F(0.5) - F(0.5^-) = 0$.

For a discrete CDF, the number of jumps (discontinuities) is at most countable — you cannot have uncountably many distinct mass points. This is a consequence of the fact that probabilities must sum to 1: uncountably many positive values can't sum to a finite number.

probability / ch07 / q01 ★ Easy

Which of the following best describes what a random variable actually is?

A
A variable that takes random values with no fixed rule
B
A deterministic function from the sample space $\Omega$ to the real line $\mathbb{R}$
C
A probability distribution defined on $[0,1]$
D
A random experiment with numerical outcomes
Answer: B

A random variable is a function — fully deterministic. Given input $\omega$, it always returns the same $X(\omega)$. The randomness comes entirely from $\omega$ being drawn at random from $\Omega$. The function itself has no randomness. The name is famously misleading. This is one of the most important conceptual clarifications in all of probability theory.
probability / ch07 / q02 ★★ Medium

$X$ is a random variable with CDF $F$. You know $F(3) = 0.7$ and $F(5) = 0.9$. What is $P(3 < X \leq 5)$?

A
$0.7 \times 0.9 = 0.63$
B
$0.7 + 0.9 = 1.6$ — impossible, so something's wrong
C
$F(5) - F(3) = 0.9 - 0.7 = 0.2$
D
$1 - F(5) = 0.1$
Answer: C — $0.2$

Property 2 of the CDF: $P(a < X \leq b) = F(b) - F(a)$. Direct application: $$P(3 < X \leq 5) = F(5) - F(3) = 0.9 - 0.7 = 0.2$$ This is the single most-used CDF formula. Every probability involving an interval can be computed from two CDF values.
probability / ch07 / q03 ★★ Medium

$X \sim \text{Uniform}[0,1]$ with CDF $F(x) = x$ for $x \in [0,1]$. What is $P(X = 0.5)$?

A
$0.5$ — it's the midpoint of $[0,1]$
B
$\frac{1}{2}$ — since the density is 1
C
$0$ — continuous RVs assign zero probability to individual points
D
Undefined — you need a density function to compute this
Answer: C — $0$

For a continuous RV, the CDF has no jumps. $P(X = c) = F(c) - F(c^-) = 0$ for every point $c$, since $F$ is continuous at $c$. Every individual value has probability exactly zero. This seems paradoxical — something must happen! But it's consistent: probabilities only make sense over intervals for continuous distributions. $P(0.49 < X \leq 0.51) = 0.02 > 0$.
probability / ch07 / q04 ★★★ Hard

A CDF $F$ has a jump of size $0.3$ at $x = 2$ and is continuous everywhere else. What does this tell you about the random variable $X$?

A
$X$ is always equal to 2
B
$P(X = 2) = 0.3$ — the jump size equals the probability mass at that point
C
$F(2) = 0.3$
D
$X$ is a continuous random variable with a spike at 2
Answer: B

The jump size of the CDF at a point $c$ is exactly $P(X = c) = F(c) - F(c^-)$. A jump of 0.3 at $x=2$ means $X$ has a probability mass of 0.3 at the point 2 — i.e., $X = 2$ with probability $30\%$. The CDF being continuous elsewhere means $X$ has no other mass points — but it could still have a continuous distribution over other regions (a mixed distribution). In finance, option payoffs often have mass at zero (the option expires worthless) plus a continuous distribution over positive values — exactly this structure.
