DTMC II — Limiting Behaviour

Why does a stationary distribution exist and is it unique? What happens to a population that keeps reproducing? What if the underlying Markov chain is hidden? This chapter answers all three — and solves the gambler's ruin problem completely.

Why Is the Stationary Distribution Unique?

In Chapter 3 we found stationary distributions by solving $\beta = \beta P$. But we never asked: could there be more than one? For an irreducible chain, the answer is no — and the proof is elegant linear algebra.

Claim: For an irreducible Markov chain on a finite state space, $\dim(\ker(P - I)) = 1$, so the stationary distribution is unique.

Proof sketch: Suppose $w$ satisfies $Pw = w$ (i.e., $w \in \ker(P-I)$). Let $i_0$ be the state where $w$ achieves its maximum: $w(i_0) = \max_i w(i)$.

Since the chain is irreducible, from $i_0$ we can reach any state $j$ — so there's a path $i_0 \to i_1 \to \cdots \to j$ with all transition probabilities positive. Suppose $w(i_1) < w(i_0)$ for some neighbour $i_1$. Then: $$w(i_0) = \sum_l P_{i_0 l}\, w(l) = P_{i_0 i_1} w(i_1) + \sum_{l \neq i_1} P_{i_0 l}\, w(l) < w(i_0)\!\sum_l P_{i_0 l} = w(i_0)$$ Contradiction. So $w(i_1) = w(i_0)$. Repeating along any path from $i_0$, all components of $w$ equal $w(i_0)$ — i.e., $w$ is a constant vector. $\ker(P-I) = \text{span}\{\mathbf{1}_n\}$, dimension 1. $\square$

By rank-nullity: $\text{rank}(P-I) = n-1$, so $\text{rank}(P-I)^T = n-1$, so $\dim\{v : v = vP\} = 1$ — unique stationary row vector.

Positive vs Null Recurrence

A recurrent state $j$ always has $P(\text{return to } j) = 1$ — but how long does it take to return?

Let $m_j = E(\text{first return time to } j \mid X_0 = j)$.

State $j$ is positive recurrent if $m_j < \infty$ — it returns in finite expected time.
State $j$ is null recurrent if $m_j = \infty$ — it eventually returns, but takes forever on average.
Key result: For an irreducible recurrent Markov chain, the stationary probability satisfies: $$\pi_j = \frac{1}{m_j}$$ If $j$ is positive recurrent: $\pi_j > 0$. If $j$ is null recurrent: $\pi_j = 0$.
Proof via SLLN: Let $T_1$ = time to first visit to $j$, $T_2$ = additional time to second visit, etc. By the strong Markov property, $T_2, T_3, \ldots$ are iid with mean $m_j$. The long-run fraction of time in state $j$ is: $$\pi_j = \lim_{n\to\infty} \frac{n}{\sum_{k=1}^n T_k} = \frac{1}{\lim_{n\to\infty} \frac{1}{n}\sum_{k=1}^n T_k} = \frac{1}{m_j}$$ where the last step uses the SLLN on the iid sequence $T_2, T_3, \ldots$ (and $T_1/n \to 0$). $\square$
🚶 Worked Example — Symmetric Random Walk is Null Recurrent

For the symmetric random walk on $\mathbb{Z}$ ($p = 1/2$), state 0 is recurrent (we proved this in Ch 3). But what is $m_0$?

Let $E(T_{i,j})$ = expected transitions from $i$ to $j$. By conditioning on the first step:

$$E(T_{0,0}) = 1 + \tfrac{1}{2}E(T_{1,0}) + \tfrac{1}{2}E(T_{-1,0})$$

By symmetry $E(T_{1,0}) = E(T_{-1,0})$. Now condition on the first step from state 1:

$$E(T_{1,0}) = 1 + \tfrac{1}{2}E(T_{2,0}) = 1 + \tfrac{1}{2}[E(T_{2,1}) + E(T_{1,0})]$$

By translation invariance: $E(T_{2,1}) = E(T_{1,0})$. So:

$$E(T_{1,0}) = 1 + E(T_{1,0}) \implies \text{no finite solution} \implies E(T_{1,0}) = \infty$$

Therefore $m_0 = \infty$ — state 0 is null recurrent. The walk returns to 0 with probability 1, but takes infinitely long on average. And $\pi_0 = 1/m_0 = 0$ — no stationary distribution exists.

Transient
$P(\text{return}) < 1$
Leaves eventually. $\sum P_{jj}^n < \infty$
Null Recurrent
$m_j = \infty$
Returns surely, but $\pi_j = 0$. Infinite state spaces only.
Positive Recurrent
$m_j < \infty$
Returns quickly. $\pi_j = 1/m_j > 0$. All finite irreducible chains.

Gambler's Ruin — The Full Solution

A gambler starts with fortune $i$ and plays against a casino. Each round: win ₹1 (prob $p$) or lose ₹1 (prob $q = 1-p$). Game ends when fortune reaches 0 (ruin) or $N$ (win). What's the probability of winning?

Let $P_i$ = probability of reaching $N$ starting from $i$. The boundary conditions are $P_0 = 0$ (already ruined) and $P_N = 1$ (already won). For $i = 1, \ldots, N-1$:

$$P_i = p\, P_{i+1} + q\, P_{i-1} \implies P_{i+1} - P_i = \frac{q}{p}(P_i - P_{i-1})$$

This is a linear recurrence with ratio $r = q/p$. Telescoping from the boundary:

$$P_i = \begin{cases} \dfrac{1-(q/p)^i}{1-(q/p)^N} & p \neq \tfrac{1}{2} \\[1em] \dfrac{i}{N} & p = \tfrac{1}{2} \end{cases}$$
When $p = 1/2$: your win probability is exactly $i/N$ — proportional to your starting fortune. Start with ₹100 against a ₹1000 target? You win 10% of the time. The game is fair, but the casino's deep pockets are the real edge.

When $p < 1/2$: as $N \to \infty$ (infinite casino), $P_i \to 0$ for any finite $i$. You are guaranteed to be ruined eventually, no matter your starting fortune. This is why casinos always win in the long run.
🎰 Worked Example — Gambler's Ruin in Numbers

A trader starts with ₹3 and plays until reaching ₹6 or going broke. Win probability per round $p = 0.4$, so $q/p = 0.6/0.4 = 1.5$.

$$P_3 = \frac{1 - (1.5)^3}{1 - (1.5)^6} = \frac{1 - 3.375}{1 - 11.39} = \frac{-2.375}{-10.39} \approx 0.229$$

Only a 22.9% chance of reaching ₹6 — even though starting halfway there. With a fair coin ($p=0.5$): $P_3 = 3/6 = 50\%$. The edge matters enormously.

Gambler's ruin is a direct model for a strategy with a daily stop-loss and profit target. If you stop trading when down ₹10,000 (ruin) or up ₹50,000 (target), and your daily edge is $p$, the formula gives your probability of hitting target before stop-loss. With $p < 0.5$, increasing your target only reduces your success probability further — a sobering reality for discretionary traders.

Hidden Markov Models (HMMs)

What if you can't observe the Markov chain directly — only noisy signals emitted by it? This is the Hidden Markov Model: a powerful framework used in speech recognition, NLP, genomics, and — crucially for us — financial regime detection.

An HMM consists of:

— A Markov chain $\{X_n\}$ with transition matrix $P$ (the hidden states)
— A signal set $S$ and emission probabilities $p(s \mid j)$ = probability of signal $s$ when in state $j$
— We observe signals $S_1, S_2, \ldots$ but not $X_1, X_2, \ldots$

The fundamental question: given observed signals $(s_1, s_2, \ldots, s_n)$, what was the most likely sequence of hidden states? Two algorithms solve the two main problems:

  • Forward algorithm: compute $P(X_n = j \mid S^n = s^n)$ — filter the current state
  • Viterbi algorithm: find the most likely state sequence — decode the hidden path

The Forward Algorithm

Define $F_n(j) = P(S^n = s^n,\, X_n = j)$ — joint probability of observed signals and being in state $j$ at time $n$. Then:

$$F_n(j) = p(s_n \mid j) \sum_i P_{ij}\, F_{n-1}(i)$$ $$F_1(j) = P(X_1 = j)\cdot p(s_1 \mid j)$$

The filtered state probability is then:

$$P(X_n = j \mid S^n = s^n) = \frac{F_n(j)}{\sum_i F_n(i)}$$
🏭 Worked Example — Production Process HMM

States: 1 = Good process, 2 = Poor process. $P = \begin{pmatrix} 0.9 & 0.1 \\ 0 & 1 \end{pmatrix}$ (once poor, stays poor).

Emissions: $p(a \mid 1) = 0.99$, $p(u \mid 1) = 0.01$, $p(a \mid 2) = 0.96$, $p(u \mid 2) = 0.04$.

Initial: $P(X_1=1) = 0.8$. Observed: $(a, u, a)$. Find $P(X_3 = 1 \mid (a,u,a))$.

Step 1: $F_1(1) = 0.8 \times 0.99 = 0.792$,   $F_1(2) = 0.2 \times 0.96 = 0.192$

Step 2 (signal $u$, $p(u|1)=0.01$, $p(u|2)=0.04$):

$$F_2(1) = 0.01\times(0.792\times0.9 + 0.192\times0) = 0.007128$$ $$F_2(2) = 0.04\times(0.792\times0.1 + 0.192\times1) = 0.010848$$

Step 3 (signal $a$, $p(a|1)=0.99$, $p(a|2)=0.96$):

$$F_3(1) = 0.99\times(0.007128\times0.9 + 0.010848\times0) \approx 0.00635$$ $$F_3(2) = 0.96\times(0.007128\times0.1 + 0.010848\times1) \approx 0.01110$$ $$P(X_3=1 \mid (a,u,a)) = \frac{0.00635}{0.00635 + 0.01110} \approx 36.4\%$$

After seeing the "unacceptable" signal in round 2, the posterior drops sharply — the process is more likely in the poor state.

In quantitative finance, the hidden states are market regimes — bull, bear, high-volatility. The "signals" are daily returns, volume, or VIX levels. A Hidden Markov Model fitted to Nifty 50 returns can estimate the probability of being in a high-volatility regime right now, even though regime is never directly observed. This drives dynamic hedging strategies.

Branching Processes

A branching process models a population where each individual independently produces a random number of offspring. Let $P_j$ = probability of producing $j$ offspring, with mean $\mu$ and variance $\sigma^2$. Start with $X_0 = 1$ individual.

$$E(X_n) = \mu^n \qquad \text{Var}(X_n) = \begin{cases} \sigma^2 \mu^{n-1} \dfrac{1-\mu^n}{1-\mu} & \mu \neq 1 \\ n\sigma^2 & \mu = 1 \end{cases}$$
Derive $E(X_n) = \mu^n$:
Given $X_{n-1}$, the $n$-th generation is $X_n = \sum_{i=1}^{X_{n-1}} Z_i$ where $Z_i$ are iid offspring counts.
$E(X_n \mid X_{n-1}) = \mu X_{n-1}$. By Tower Property: $E(X_n) = \mu\, E(X_{n-1})$.
With $E(X_0) = 1$: $E(X_n) = \mu^n$. $\square$

Extinction Probability

Let $\pi_0 = P(\text{population eventually dies out} \mid X_0 = 1)$. This is determined by $\mu$:

Extinction Theorem:
— If $\mu \le 1$: $\pi_0 = 1$ — extinction is certain.
— If $\mu > 1$: $\pi_0 < 1$ — extinction is possible but not certain. $\pi_0$ is the smallest non-negative root of $\pi_0 = \phi(\pi_0)$, where $\phi(z) = \sum_{j=0}^\infty P_j z^j$ is the probability generating function.
🌱 Worked Example — When Does a Lineage Die Out?

Case 1: $P_0 = 1/2$, $P_1 = 1/8$, $P_2 = 3/8$. Mean $\mu = 0 \cdot \frac{1}{2} + 1 \cdot \frac{1}{8} + 2 \cdot \frac{3}{8} = \frac{7}{8} < 1$. So $\pi_0 = 1$ — extinction is certain.

Case 2: $P_0 = 3/8$, $P_1 = 1/8$, $P_2 = 1/2$. Mean $\mu = 0 \cdot \frac{3}{8} + 1 \cdot \frac{1}{8} + 2 \cdot \frac{1}{2} = \frac{9}{8} > 1$. Solve $\pi_0 = \phi(\pi_0)$:

$$\pi_0 = \frac{3}{8} + \frac{1}{8}\pi_0 + \frac{1}{2}\pi_0^2 \implies 4\pi_0^2 - 7\pi_0 + 3 = 0 \implies \pi_0 = \frac{3}{4} \text{ or } 1$$

The smallest non-negative root is $\pi_0 = 3/4$ — even with a supercritical mean, there's a 75% chance of eventual extinction.

Branching processes model startup survival, viral spread, and — in finance — the propagation of defaults in interconnected networks. If each defaulting firm causes on average $\mu$ other firms to default, the cascade dies out surely when $\mu \le 1$ and may explode when $\mu > 1$. The 2008 financial crisis saw $\mu > 1$ in the credit default swap network — a supercritical branching process of contagion.

An ergodic Markov chain has stationary distribution $\pi = [0.25, 0.50, 0.25]$. What is the mean return time to state 0?

A
$0.25$ steps
B
$4$ steps
C
$2$ steps
D
$1$ step
Answer: B — 4 steps

$\pi_j = 1/m_j \implies m_j = 1/\pi_j$. For state 0: $m_0 = 1/0.25 = 4$ steps.

Low stationary probability = long mean return time. State 0 is visited only 25% of the time, so on average you wait 4 steps between visits.

A trader starts with ₹2 and targets ₹4. Each trade wins or loses ₹1 with equal probability ($p = 1/2$). What is the probability of reaching the target before going broke?

A
$1/4$
B
$3/4$
C
$1/2$
D
$2/3$
Answer: C — $1/2$

With $p = 1/2$, the gambler's ruin formula gives $P_i = i/N$. Here $i = 2$, $N = 4$: $$P_2 = \frac{2}{4} = \frac{1}{2}$$ Starting halfway to the target with a fair coin gives exactly 50/50 odds. With $p=1/2$, win probability equals starting fraction of total. It's beautifully symmetric.
stochastic / ch04 / q03 ★★ Medium

A branching process has $P_0 = 1/3$, $P_1 = 1/3$, $P_2 = 1/3$. What is the mean number of offspring $\mu$, and is extinction certain?

A
$\mu = 1$, extinction certain
B
$\mu = 2/3$, extinction certain
C
$\mu = 1$, extinction is possible but not certain ($\pi_0 < 1$)
D
$\mu = 1$, extinction certain since $\mu \le 1$
Answer: C — $\mu = 1$, but extinction is certain since $\mu \le 1$

Wait — the theorem says if $\mu \le 1$, extinction is certain ($\pi_0 = 1$). So actually D and C conflict. Let me clarify:

$\mu = 0 \cdot \frac{1}{3} + 1 \cdot \frac{1}{3} + 2 \cdot \frac{1}{3} = 1$. When $\mu = 1$ (critical case), extinction IS certain — $\pi_0 = 1$. The correct answer here is: $\mu = 1$ and extinction is certain.

The critical case $\mu = 1$ is subtle: the population neither grows nor shrinks on average, but still goes extinct almost surely — just very slowly. This is the Galton-Watson result for critical branching processes.
stochastic / ch04 / q04 ★★ Medium

In the HMM forward algorithm, $F_n(j)$ computes which probability?

A
$P(X_n = j)$ — the marginal state probability at time $n$
B
$P(X_n = j \mid S^n = s^n)$ — the filtered state probability
C
$P(S^n = s^n,\, X_n = j)$ — the joint probability of signals and state
D
$P(S_n = s_n \mid X_n = j)$ — the emission probability
Answer: C

$F_n(j) = P(S^n = s^n, X_n = j)$ — the joint probability of the entire observed signal sequence up to time $n$ AND being in state $j$ at time $n$.

Option B (the filtered probability) is obtained by normalising: $P(X_n=j \mid S^n=s^n) = F_n(j) / \sum_i F_n(i)$. Option D is just the emission probability $p(s_n \mid j)$ — one factor in the recursion, not the whole thing. The forward algorithm computes joint probabilities; normalising at the end gives the posterior.
stochastic / ch04 / q05 ★★★ Hard

A branching process has $P_0 = 1/4$, $P_2 = 3/4$ (no single offspring). The extinction probability $\pi_0$ satisfies $\pi_0 = \phi(\pi_0)$. Which equation is correct, and what is $\pi_0$?

A
$\pi_0 = 1/4 + 3/4\,\pi_0$ → $\pi_0 = 1$
B
$\pi_0 = 1/4 + 3/4\,\pi_0^2$ → $\pi_0 = 1/3$
C
$\pi_0 = 3/4 + 1/4\,\pi_0^2$ → $\pi_0 = 1$
D
$\pi_0 = 1/4\,\pi_0 + 3/4\,\pi_0^2$ → $\pi_0 = 1/3$
Answer: B — $\pi_0 = 1/3$

The PGF is $\phi(z) = \sum_j P_j z^j = P_0 z^0 + P_2 z^2 = \frac{1}{4} + \frac{3}{4}z^2$.
Setting $\phi(\pi_0) = \pi_0$: $\frac{1}{4} + \frac{3}{4}\pi_0^2 = \pi_0 \implies 3\pi_0^2 - 4\pi_0 + 1 = 0 \implies (3\pi_0 - 1)(\pi_0 - 1) = 0$
Roots: $\pi_0 = 1/3$ and $\pi_0 = 1$. The smallest non-negative root is $\mathbf{1/3}$.

Check: $\mu = 0 \times \frac{1}{4} + 2 \times \frac{3}{4} = \frac{3}{2} > 1$ — supercritical, so extinction probability $= 1/3 < 1$. There's a 33% chance of extinction and 67% chance the population grows forever.
