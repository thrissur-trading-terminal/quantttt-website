Discrete Distributions

Three distributions — Bernoulli, Binomial, Poisson — power an enormous fraction of probability and statistics. Each arises from a single clean idea, and each has a remarkable reproductive property: two independent copies sum to a distribution of the same family.

Discrete Random Variables & the PMF

A random variable $X$ is discrete if it takes at most countably many values $x_1, x_2, x_3, \ldots$ on the real line, with each value having positive probability $p_i > 0$ and $\sum_i p_i = 1$. The function $p(x_i) = P(X = x_i)$ is the Probability Mass Function (PMF).

Think of the PMF as a bar chart: each mass point gets a vertical bar whose height equals its probability. The bars must sum to 1. Bernoulli has 2 bars. Binomial has $n+1$ bars. Poisson has infinitely many bars — and they still sum to 1, courtesy of the Taylor series of $e^\lambda$.

1. Bernoulli($p$)

$X$ takes exactly two values — 0 and 1 — with $P(X=0) = 1-p$ and $P(X=1) = p$, where $0 < p < 1$. Every binary outcome is Bernoulli: a coin toss, an option expiring in-the-money, a credit default event, an item being non-defective. The Bernoulli is the atom from which all discrete distributions are built.

2. Binomial($n$, $p$)

Run $n$ independent Bernoulli($p$) trials. Let $X$ = number of successes. Then: $$P(X=k) = \binom{n}{k} p^k q^{n-k}, \quad k = 0, 1, \ldots, n, \quad q = 1-p$$ Why this formula? $\binom{n}{k}$ ways to choose which $k$ positions are successes; $p^k q^{n-k}$ for each arrangement by independence. Why sum to 1? $(q+p)^n = 1$ and the binomial theorem gives exactly $\sum_{k=0}^n \binom{n}{k}p^k q^{n-k}$. The Binomial distribution is the binomial theorem in probability clothing.
Reproductive Property: If $X \sim \text{Bin}(n,p)$ and $Y \sim \text{Bin}(m,p)$ are independent (same $p$), then: $$X + Y \sim \text{Bin}(m+n,\, p)$$
The event $X+Y=k$ decomposes into $k+1$ disjoint cases $(X=j,\, Y=k-j)$: $$P(X+Y=k) = \sum_{j=0}^{k} \binom{n}{j}p^j q^{n-j} \cdot \binom{m}{k-j}p^{k-j}q^{m-k+j} = p^k q^{m+n-k}\sum_{j=0}^{k}\binom{n}{j}\binom{m}{k-j}$$ By the Vandermonde identity: $\displaystyle\sum_{j=0}^{k}\binom{n}{j}\binom{m}{k-j} = \binom{m+n}{k}$, giving: $$P(X+Y=k) = \binom{m+n}{k}p^k q^{m+n-k} \qquad \square$$
The same $p$ is essential. If $p_1 \neq p_2$, the Vandermonde factorisation breaks because $p^k q^{n+m-k}$ doesn’t factor cleanly — and $X+Y$ exits the Binomial family entirely.

3. Poisson($\lambda$)

$X$ takes all non-negative integers $0, 1, 2, \ldots$ with: $$P(X=k) = \frac{e^{-\lambda}\,\lambda^k}{k!}, \quad k = 0,1,2,\ldots, \quad \lambda > 0$$ Why sum to 1? The Taylor series $e^\lambda = \sum_{k=0}^\infty \frac{\lambda^k}{k!}$, so $\sum_{k=0}^\infty \frac{e^{-\lambda}\lambda^k}{k!} = e^{-\lambda} \cdot e^{\lambda} = 1$. The Poisson is the exponential function wearing a probability hat.
Reproductive Property: If $X \sim \text{Poi}(\lambda)$ and $Y \sim \text{Poi}(\mu)$ are independent, then: $$X + Y \sim \text{Poi}(\lambda + \mu)$$ No matching condition required — any two independent Poissons merge cleanly.
$$P(X+Y=k) = \sum_{j=0}^k e^{-\lambda}\frac{\lambda^j}{j!}\cdot e^{-\mu}\frac{\mu^{k-j}}{(k-j)!} = \frac{e^{-(\lambda+\mu)}}{k!}\sum_{j=0}^k \binom{k}{j}\lambda^j\mu^{k-j}$$ By the binomial theorem: $\displaystyle\sum_{j=0}^k\binom{k}{j}\lambda^j\mu^{k-j} = (\lambda+\mu)^k$, giving: $$P(X+Y=k) = \frac{e^{-(\lambda+\mu)}(\lambda+\mu)^k}{k!} \qquad \square$$
The Poisson distribution models rare-event counts in Indian markets: NSE circuit breaker triggers per month, SEBI enforcement actions per quarter, Nifty moves exceeding 3 standard deviations per year. The reproductive property means you can aggregate Poisson processes across desks or time periods and stay in the Poisson family — an enormous practical convenience for risk aggregation.

probability / ch08 / q01★ Easy

A fair coin is tossed 5 times. What is $P(X=3)$ where $X$ = number of heads?

A
$3/5$
B
$10/32 = 5/16$
C
$1/8$
D
$3/32$
Answer: B — $5/16$

$X \sim \text{Bin}(5,0.5)$. $P(X=3) = \binom{5}{3}(0.5)^5 = 10/32 = 5/16$. $\binom{5}{3}=10$ counts the arrangements of 3 heads in 5 tosses. Each specific sequence has probability $(0.5)^5 = 1/32$.
probability / ch08 / q02★★ Medium

$X \sim \text{Bin}(4,0.3)$ and $Y \sim \text{Bin}(6,0.3)$ are independent. What is the distribution of $X+Y$?

A
Bin(10, 0.09) — multiply the probabilities
B
Bin(10, 0.3) — add trial counts, keep $p$
C
Bin(10, 0.6) — add both parameters
D
Cannot be determined without more info
Answer: B — Bin(10, 0.3)

Reproductive property: same $p=0.3$, so $n$-parameters add: $4+6=10$. Think of it as one 10-trial experiment with the same coin. Option C incorrectly adds $p$ values — that would give a probability exceeding 1 eventually.
probability / ch08 / q03★★ Medium

Errors in a system occur at rate $\lambda=2$ per hour (Poisson). What is $P(\text{no errors in one hour})$?

A
$0$
B
$e^{-2} \approx 0.135$
C
$1/2$
D
$2e^{-2} \approx 0.271$
Answer: B

$P(X=0) = e^{-2} \cdot 2^0 / 0! = e^{-2} \approx 0.135$. About 13.5% chance of a quiet hour. Option D is $P(X=1) = 2e^{-2}$ — exactly one error, not zero.
probability / ch08 / q04★★★ Hard

Orders arrive from two independent desks: Desk A at $\lambda=3$/min and Desk B at $\mu=5$/min (both Poisson). What is $P(\text{exactly 4 total orders in one minute})$?

A
$P(\text{Poi}(3){=}4) + P(\text{Poi}(5){=}4)$
B
$P(\text{Poi}(8){=}4) = \dfrac{8^4 e^{-8}}{4!}$
C
$P(\text{Poi}(15){=}4)$
D
$P(\text{Poi}(3){=}4) \times P(\text{Poi}(5){=}4)$
Answer: B

Reproductive property: total $\sim \text{Poi}(3+5) = \text{Poi}(8)$. $P(\text{total}=4) = \frac{e^{-8} \cdot 8^4}{24} \approx 0.057$. Rates add. Option A adds probabilities (wrong). Option D gives $P(A=4 \text{ AND } B=4)$, not $P(A+B=4)$.
probability / ch08 / q05★★★ Hard

The Binomial reproductive property requires the same $p$. The Poisson property has no such constraint. Why?

A
Both properties actually work without restrictions
B
Poisson has no upper bound while Binomial does
C
Binomial has two parameters $(n,p)$ and only $n$ adds; different $p$ exits the family. Poisson has one parameter $\lambda$ which simply adds
D
Poisson is a special case of Binomial and inherits its restrictions
Answer: C

With different $p$ values, $p^k q^{n+m-k}$ does not factor from the Vandermonde sum — the result is not Binomial. For Poisson: $e^{-\lambda} \cdot e^{-\mu} = e^{-(\lambda+\mu)}$ always, no condition needed. One-parameter families are cleaner to combine than two-parameter ones.
