Functions of Random Variables

If $X$ has a known distribution, what is the distribution of $Y = h(X)$? This chapter develops the systematic answer: the change-of-variable theorem for monotonic functions, the non-monotonic extension, and the beautiful Probability Integral Transform — which says that $F(X)$ is always Uniform$(0,1)$, no matter what distribution $X$ follows.

Discrete Case

If $h(x_{i_1}), h(x_{i_2}), \ldots, h(x_{i_m})$ all equal $y_j$, then $P(Y=y_j) = \sum_{k=1}^m P(X=x_{i_k})$.

Example — $X\sim\text{Bin}(10,\frac{1}{2})$, $Y=3X+1$

$Y$ takes values $1,4,7,\ldots,31$. Find $P(Y\leq 10)$:

$Y\leq 10 \iff 3X+1\leq 10 \iff X\leq 3$. So:

$$P(Y\leq 10) = \sum_{x=0}^3\binom{10}{x}\left(\frac{1}{2}\right)^{10} = \frac{1+10+45+120}{1024} = \frac{176}{1024} = \frac{11}{64}$$

Continuous Case: The CDF Method

To find the PDF of $Y=h(X)$: (1) find $G(y)=P(Y\leq y)$ by translating to an event about $X$, then (2) differentiate to get $g(y) = G'(y)$.

Example 1 — Increasing: $f(x)=2x$ on $[0,1]$, $Y=3X+1$

$G(y) = P(3X+1\leq y) = P\!\left(X\leq\frac{y-1}{3}\right) = \int_0^{(y-1)/3}2x\,dx = \left(\frac{y-1}{3}\right)^2$, valid for $1\leq y\leq 4$.

$$g(y) = G'(y) = \frac{2(y-1)}{9}, \quad 1\leq y\leq 4$$
Example 2 — Decreasing: same $f(x)$, $Y=e^{-X}$

$h(x)=e^{-x}$ is decreasing. $P(e^{-X}\leq y) = P(-X\leq\ln y) = P(X\geq -\ln y) = 1-F(-\ln y)$.

Since $y=e^{-x}\in[e^{-1},1]$ for $x\in[0,1]$:

$$G(y) = 1-(-\ln y)^2 \implies g(y) = -2\ln y\cdot\frac{1}{y} = \frac{-2\ln y}{y}, \quad e^{-1}\leq y\leq 1$$

(valid since $\ln y < 0$ on this interval, making $g(y) > 0$).

The Change-of-Variable Theorem

Suppose $X$ has PDF $f(x) > 0$ on $(a,b)$, and $h(x)$ is strictly monotonic and differentiable on $(a,b)$. Let $Y=h(X)$. Then: $$\boxed{g(y) = f\!\left(h^{-1}(y)\right)\left|\frac{dx}{dy}\right|}$$ where $x = h^{-1}(y)$ and $dx/dy$ is the derivative of the inverse function.

Support: if $h$ is increasing, $y\in(h(a),h(b))$; if decreasing, $y\in(h(b),h(a))$.
Case 1 (increasing): $G(y) = P(h(X)\leq y) = P(X\leq h^{-1}(y)) = F(h^{-1}(y))$. Differentiating: $g(y) = f(h^{-1}(y))\cdot\frac{d}{dy}h^{-1}(y) = f(x)\cdot\frac{dx}{dy}$. Here $dx/dy > 0$. $\square$

Case 2 (decreasing): $G(y) = P(X\geq h^{-1}(y)) = 1-F(h^{-1}(y))$. Differentiating: $g(y) = -f(x)\cdot\frac{dx}{dy}$. Here $dx/dy < 0$, so $g(y) = f(x)|dx/dy|$. $\square$

Both cases give $g(y) = f(x)|dx/dy|$, i.e. the absolute value handles the sign automatically.

Non-Monotonic Case: $Y = X^2$

When $h$ is not monotonic, we cannot use a single inverse. Instead, work directly with the CDF.

$G(y) = P(X^2\leq y) = P(-\sqrt{y}\leq X\leq\sqrt{y}) = F(\sqrt{y})-F(-\sqrt{y})$. $$g(y) = f(\sqrt{y})\cdot\frac{1}{2\sqrt{y}} + f(-\sqrt{y})\cdot\frac{1}{2\sqrt{y}} = \frac{f(\sqrt{y})+f(-\sqrt{y})}{2\sqrt{y}}$$
✓ Verification: $X\sim N(0,1)$, $Y=X^2$

$f(x) = \frac{1}{\sqrt{2\pi}}e^{-x^2/2}$. Since $f(\sqrt{y}) = f(-\sqrt{y})$ (even function):

$$g(y) = \frac{2f(\sqrt{y})}{2\sqrt{y}} = \frac{1}{\sqrt{2\pi}}e^{-y/2}\cdot\frac{1}{\sqrt{y}} = \frac{(1/2)^{1/2}}{\Gamma(1/2)}e^{-y/2}y^{1/2-1}$$

This is $\Gamma(\frac{1}{2},\frac{1}{2}) = \chi^2_{(1)}$. ✓ Consistent with our earlier result.

Cauchy$^2$: $X\sim C(1,0)$, $Y=X^2$

$f(x)=\frac{1}{\pi}\frac{1}{1+x^2}$. Since $f(\sqrt{y})=f(-\sqrt{y})$:

$$g(y) = \frac{1}{\pi}\frac{1}{1+y}\cdot\frac{1}{\sqrt{y}} = \frac{1}{\pi}\cdot\frac{y^{1/2-1}}{1+y} = \frac{1}{B(1/2,1/2)}\cdot\frac{y^{1/2-1}}{(1+y)^{1/2+1/2}}$$

Since $B(\frac{1}{2},\frac{1}{2})=\pi$: $Y = X^2 \sim \beta_2(\frac{1}{2},\frac{1}{2})$.

$X\sim\text{Exp}(1)$ (positive), $Y=X^2$: apply theorem directly

On $(0,\infty)$, $h(x)=x^2$ is increasing. $h^{-1}(y)=\sqrt{y}$, $|dx/dy|=1/(2\sqrt{y})$.

$$g(y) = e^{-\sqrt{y}}\cdot\frac{1}{2\sqrt{y}} = \frac{1}{2}y^{1/2-1}e^{-y^{1/2}}, \quad y>0$$

This is the Weibull distribution (with $c=\frac{1}{2}$).

The Probability Integral Transform

Let $X$ be any continuous RV with CDF $F(x)$. Define $Y = F(X)$. Then: $$Y \sim U(0,1)$$ This holds for every continuous distribution — Normal, Exponential, Cauchy, anything.
Since $F$ is strictly increasing (continuous $f(x) > 0$): $y=F(x) \Rightarrow dy/dx = f(x)$, so $|dx/dy| = 1/f(x)$. Applying the change-of-variable theorem: $$g(y) = f(x)\cdot\frac{1}{f(x)} = 1, \quad y\in(0,1) \qquad \square$$ Alternative proof: $P(Y\leq y) = P(F(X)\leq y) = P(X\leq F^{-1}(y)) = F(F^{-1}(y)) = y$. So $G(y) = y$, hence $g(y) = 1$ — Uniform$(0,1)$. $\square$
The Probability Integral Transform is the foundation of Monte Carlo simulation. To generate samples from any distribution $F$: (1) generate $U\sim U(0,1)$, (2) return $X = F^{-1}(U)$. The PIT guarantees $X\sim F$. Every risk simulation on NSE — generating random Nifty return paths, stochastic volatility models, option pricing by Monte Carlo — uses this result under the hood.

$X \sim U(0,1)$ with $f(x)=1$. Find the PDF of $Y = -\ln X$.

A
$g(y) = e^y$, $y \in (-\infty, 0)$
B
$g(y) = e^{-y}$, $y > 0$ — Exp(1)
C
$g(y) = 1/y$, $y \in (0,1)$
D
$g(y) = \ln y$, $y > 1$
Answer: B — Exp(1)

$h(x) = -\ln x$ is decreasing on $(0,1)$. $h^{-1}(y) = e^{-y}$, $|dx/dy| = e^{-y}$. $g(y) = f(e^{-y})\cdot e^{-y} = 1\cdot e^{-y} = e^{-y}$ for $y>0$. This is the Exp(1) PDF — a beautiful connection: the negative log of a Uniform is Exponential. This is the PIT in reverse: if $U\sim U(0,1)$ then $-\ln U \sim \text{Exp}(1)$.

$X \sim \text{Exp}(1)$, $f(x) = e^{-x}$ for $x > 0$. What is the PDF of $Y = 2X$?

A
$g(y) = e^{-y}$, $y > 0$ — scaling doesn't change the distribution
B
$g(y) = \frac{1}{2}e^{-y/2}$, $y > 0$ — Exp($\frac{1}{2}$)
C
$g(y) = 2e^{-2y}$, $y > 0$ — Exp(2)
D
$g(y) = e^{-y/2}$, $y > 0$
Answer: B — Exp$(\frac{1}{2})$

$h(x)=2x$, $h^{-1}(y)=y/2$, $|dx/dy|=1/2$. $g(y) = e^{-y/2}\cdot\frac{1}{2} = \frac{1}{2}e^{-y/2}$ for $y>0$. Doubling $X$ halves the rate $\lambda$: if $X\sim\text{Exp}(1)$ then $2X\sim\text{Exp}(\frac{1}{2})$. Mean doubles from 1 to 2, as expected.
probability / ch19 / q03★★ Medium

$X \sim N(\mu, \sigma^2)$. $Y = F(X)$ where $F$ is the CDF of $X$. What is the distribution of $Y$?

A
$N(0,1)$ — $F(X)$ standardises $X$
B
$N(\mu, \sigma^2)$ — same as $X$
C
$U(0,1)$ — the Probability Integral Transform
D
Depends on $\mu$ and $\sigma^2$
Answer: C — $U(0,1)$

The Probability Integral Transform: for any continuous RV $X$ with CDF $F$, $Y=F(X)\sim U(0,1)$. The distribution of $X$ is completely irrelevant — Normal, Exponential, Cauchy, it makes no difference. This is not standardisation (which gives $N(0,1)$, not Uniform); it's a fundamentally different operation.
probability / ch19 / q04★★★ Hard

$X \sim U(0,1)$, $f(x)=1$. Find the PDF of $Y = X^2$ using the non-monotonic formula $g(y) = \frac{f(\sqrt{y})+f(-\sqrt{y})}{2\sqrt{y}}$.

A
$g(y) = \frac{1}{\sqrt{y}}$, $y\in(0,1)$ — using the full formula
B
$g(y) = \frac{1}{2\sqrt{y}}$, $y\in(0,1)$ — only one branch contributes
C
$g(y) = 2\sqrt{y}$, $y\in(0,1)$
D
$g(y) = 1$, $y\in(0,1)$ — Uniform is preserved under squaring
Answer: B — $\frac{1}{2\sqrt{y}}$

$X\in(0,1)$, so $-\sqrt{y}\notin(0,1)$ and $f(-\sqrt{y})=0$. Only the positive branch contributes: $g(y) = \frac{f(\sqrt{y})}{2\sqrt{y}} = \frac{1}{2\sqrt{y}}$ for $y\in(0,1)$. Equivalently via the monotone theorem: $h(x)=x^2$ is increasing on $(0,1)$, $h^{-1}(y)=\sqrt{y}$, $|dx/dy|=1/(2\sqrt{y})$, giving the same answer. Check: $\int_0^1\frac{1}{2\sqrt{y}}dy = [\sqrt{y}]_0^1 = 1$. ✓
probability / ch19 / q05★★★ Hard

To simulate Nifty log-returns from a $N(\mu,\sigma^2)$ distribution using a $U(0,1)$ random number generator, which formula gives a correctly distributed sample $X$?

A
$X = U$ directly — Uniform approximates Normal for large samples
B
$X = \mu + \sigma U$ — shift and scale the Uniform
C
$X = F^{-1}(U)$ where $F$ is the $N(\mu,\sigma^2)$ CDF — inverse transform method
D
$X = -\ln U$ — the log transform
Answer: C — Inverse transform sampling

By the PIT in reverse: if $U\sim U(0,1)$, then $F^{-1}(U)\sim F$ for any CDF $F$. Set $F(X)=U$, i.e. $X=F^{-1}(U)$, then $X\sim N(\mu,\sigma^2)$. Option D ($-\ln U$) gives Exp(1), not Normal. Option B gives a Uniform shifted and scaled — still not Normal. This inverse transform method is the fundamental Monte Carlo sampling algorithm.
