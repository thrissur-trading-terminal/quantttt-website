Order Statistics — Applications

Three applications of order statistics: computing moments of min/max from $U(0,1)$, the joint PDF of two order statistics, and the deep Exponential spacing theorem — which shows that successive gaps $X_{(r+1)}-X_{(r)}$ are independent Exponentials, yielding a harmonic-series formula for the expected range.

Min and Max of $U(0,1)$: Moments

$X_1,\ldots,X_n$ iid $U(0,1)$. Find $E(X_{(1)})$ and $E(X_{(n)})$.

Minimum: $f_{(1)}(x) = n(1-x)^{n-1}$. This is $\beta_1(1,n)$. Directly:

$$E(X_{(1)}) = \int_0^1 x\cdot n(1-x)^{n-1}\,dx = n\cdot B(2,n) = n\cdot\frac{1!\,(n-1)!}{(n+1)!} = \frac{1}{n+1}$$

Maximum: $f_{(n)}(x) = nx^{n-1}$. This is $\beta_1(n,1)$.

$$E(X_{(n)}) = \int_0^1 x\cdot nx^{n-1}\,dx = n\int_0^1 x^n\,dx = \frac{n}{n+1}$$

Expected Range:

$$E(X_{(n)}-X_{(1)}) = \frac{n}{n+1}-\frac{1}{n+1} = \frac{n-1}{n+1}$$
General formula for $U(0,1)$: $E(X_{(r)}) = r/(n+1)$. The $n+1$ equally spaced quantile points $\frac{1}{n+1}, \frac{2}{n+1}, \ldots, \frac{n}{n+1}$ are the expected order statistics of a $U(0,1)$ sample of size $n$.

Joint PDF of Two Order Statistics

For $1\leq r < s\leq n$, the joint PDF of $(X_{(r)}, X_{(s)})$ at $(x,y)$ with $x < y$ is: $$f_{r,s}(x,y) = \frac{n!}{(r-1)!(s-r-1)!(n-s)!}[F(x)]^{r-1}[F(y)-F(x)]^{s-r-1}[1-F(y)]^{n-s}f(x)f(y)$$ Intuition: $r-1$ observations below $x$, one at $x$, $s-r-1$ between $x$ and $y$, one at $y$, $n-s$ above $y$.
Verify Range Formula: $U(0,1)$ via Joint PDF of Min and Max

For $r=1$, $s=n$: $s-r-1 = n-2$ observations between min and max.

$$f_{1,n}(x,y) = n(n-1)(y-x)^{n-2}\cdot 1\cdot 1 = n(n-1)(y-x)^{n-2}, \quad 0$E(X_{(n)}-X_{(1)})$: substitute $g(x,y)=y-x$:

$$= n(n-1)\int_0^1\int_x^1(y-x)^{n-1}\,dy\,dx = n(n-1)\int_0^1\frac{(1-x)^n}{n}\,dx = (n-1)\cdot\frac{1}{n+1} = \frac{n-1}{n+1}\checkmark$$

Exponential Spacings: A Deep Theorem

If $X_1,\ldots,X_n$ are iid $\text{Exp}(\lambda)$, then for each $r=1,\ldots,n-1$:

(1) $X_{(r)}$ and $X_{(r+1)}-X_{(r)}$ are independent.

(2) $X_{(r+1)}-X_{(r)} \sim \text{Exp}((n-r)\lambda)$.

Expected Range of Exp$(\lambda)$ Sample: Harmonic Sum

Telescoping to prove $E(\text{Range}) = \frac{1}{\lambda}\left(1+\frac{1}{2}+\cdots+\frac{1}{n-1}\right)$

Telescope the range using successive spacings:

$$X_{(n)}-X_{(1)} = \sum_{r=1}^{n-1}(X_{(r+1)}-X_{(r)})$$

By linearity of expectation and the spacing theorem:

$$E(X_{(n)}-X_{(1)}) = \sum_{r=1}^{n-1}E(X_{(r+1)}-X_{(r)}) = \sum_{r=1}^{n-1}\frac{1}{(n-r)\lambda} = \frac{1}{\lambda}\sum_{r=1}^{n-1}\frac{1}{n-r} = \frac{1}{\lambda}\left(1+\frac{1}{2}+\cdots+\frac{1}{n-1}\right)$$ Verification:
$n=3$: $E(\text{Range}) = \frac{1}{\lambda}(1+\frac{1}{2}) = \frac{3}{2\lambda}$ ✓ (matches direct calculation)
$n=4$: $E(\text{Range}) = \frac{1}{\lambda}(1+\frac{1}{2}+\frac{1}{3}) = \frac{11}{6\lambda}$ ✓
The harmonic sum $H_{n-1} = 1+\frac{1}{2}+\cdots+\frac{1}{n-1}$ governs how the expected range of iid Exponential observations grows with sample size. In order-flow analysis on NSE: if institutional orders arrive as a Poisson process (iid Exp inter-arrivals), the expected spread between the earliest and latest order in a batch of $n$ grows as $H_{n-1}/\lambda$. This models the "order footprint" — how detectably spread out a large institutional execution is in time.

$X_1,\ldots,X_9$ iid $U(0,1)$. What is $E(X_{(9)})$, the expected maximum?

A
$9/10$
B
$9/10$ — using $E(X_{(r)}) = r/(n+1) = 9/10$
C
$8/9$
D
$1$ — the maximum of many uniforms approaches 1
Answer: B — $9/10$

$E(X_{(r)}) = r/(n+1)$. With $r=n=9$ and $n+1=10$: $E(X_{(9)}) = 9/10$. Answers A and B say the same thing — B is correct for the right reason. The formula is $r/(n+1)$, not $r/n$. Option D is the limit as $n\to\infty$ but not a finite-$n$ expected value.

$X_1,\ldots,X_n$ iid $U(0,1)$. What is the expected range $E(X_{(n)}-X_{(1)})$?

A
$1 - 1/n$
B
$(n-1)/(n+1)$
C
$(n-1)/n$
D
$n/(n+1)$
Answer: B — $(n-1)/(n+1)$

$E(\text{Range}) = E(X_{(n)})-E(X_{(1)}) = \frac{n}{n+1}-\frac{1}{n+1} = \frac{n-1}{n+1}$. For $n=2$: expected range $=1/3$. For $n=10$: $9/11\approx 0.82$. As $n\to\infty$: expected range $\to 1$ (makes sense — more samples cover more of $[0,1]$).
probability / ch23 / q03★★ Medium

$X_1,\ldots,X_n$ iid Exp$(\lambda)$. The spacing $X_{(r+1)}-X_{(r)}$ follows which distribution?

A
Exp$(\lambda)$ — same as the original variables
B
Exp$(r\lambda)$ — depends on the position $r$
C
Exp$((n-r)\lambda)$ — depends on how many observations are above
D
Exp$(n\lambda)$ — same as the minimum
Answer: C — Exp$((n-r)\lambda)$

The spacing theorem: $X_{(r+1)}-X_{(r)}\sim\text{Exp}((n-r)\lambda)$. Intuition: given that $X_{(r)}=x$, there are $n-r$ observations still above $x$, competing to be the next arrival. The minimum of $n-r$ iid Exp$(\lambda)$ variables above $x$ is Exp$((n-r)\lambda)$. This is the memoryless property in action.

probability / ch23 / q05★★★ Hard

For the Exponential spacing theorem, the joint PDF of $(X_{(r)}, X_{(r+1)}-X_{(r)})$ factorises. What does this factorisation prove?

A
That $X_{(r)}$ and $X_{(r+1)}$ have the same distribution
B
That the range is always Exponential
C
That $X_{(r)}$ and $X_{(r+1)}-X_{(r)}$ are independent, and identifies the marginal distribution of each
D
That the joint PDF equals zero outside the support
Answer: C

If $f_{U,V}(u,v) = g(v)\cdot h(u)$, then by definition $U$ and $V$ are independent with marginal PDFs proportional to $g$ and $h$. In this case, $g(v)$ is exactly the PDF of $X_{(r)}\sim\Gamma(\lambda,r)$... wait, it's the PDF of the Exp $r$th order statistic. And $h(u)=(n-r)\lambda e^{-(n-r)\lambda u}$ is Exp$((n-r)\lambda)$. This is the standard criterion for independence: joint = product of marginals.
